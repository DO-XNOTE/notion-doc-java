其实它的一个英文单词是 Nginx，也就是 x 引擎，那么它是一个高性能的 http 和反向代理 web 服务器，同时它也能够提供一些 imap、 pop 3 以及是 SMTP 服务。那么这三个其实是和邮件相关的内容。在我们课程里面我们主要是以反商代理为主，这个是引用自百度百科的。那么其实恩迪斯它是由俄罗斯的一个哥们所开发的一开源的服务器，性能是非常非常高的，那么它主要的功能就是一个反向代理，通过反向代理可以去实现集群以及是负载均衡，另外还有是 UL 的路由等等相应的控制都是可以去实现的，通过它的一个配置文件，我们就可以去实现集群和复杂均衡实现起来它的配置是十分的简单的。并且它也可以去实现一个热加载，你不需要去重启服务器，以免我们的一个用户请求的丢失，所以乐加载也是非常不错的一点。
