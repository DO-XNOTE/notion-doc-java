那么首先在这个文件里面，整个文件它就是一个面，那么它是一个全局的内容，全局的配置我们都是可以配置在一个整体的文件里面的，那么这个 man 的话，我们又可以把它称之为是一个容器。随后我们会有一个event，那么这个 event 我们之前也说了，它是用于去配置工作模式以及是连接处的，一个是依破，一个是connections，那么相应的内容是配置到 Innet 里面，那么这个 event 在这个地方我们是称之为它是一个指定块，那么这个指定块的话我们是需要把它放在我们这个 man 这个全局配置里面去的。


那么随后下方就是一个HTTP，那么这个就是和 HTTP 相关的一些配置，那么 HTTP 它本身就是一个模块，那么在这个 HTTP 里面它又会有一个server，那么 server 的话我们之前说了，那么它是一个虚拟主机，它也是一个服务器，我们是可以去做一个相应的配置的。


那么在默认的这个 config 配置文件里面，它其实就会有一台server，那么它显示的内容就是我们之前在浏览器里面所展示的这个内容，其实就是这一块内容，它是默认为我们去配的。当然我们也可以在这个 h t p 里面去配置多个不同的server，作为不同的虚拟组织也是没有问题的。那么随后在这个 server 里面我们又会有一个 Lo 科系，那么这个 Lo 科系的话它是一个路由规则，它里面可以包含很多的表达式。那么随后的话我们还会有一个upstream，这个 Upstream 我们也是配置在 HTTP 里面的，这个的话我们其实还没有接触到，那么它是用于去配置集群以及是内网服务器相关的内容都是配置到这里面的。另外负载均衡的规则也是放在 upstream 里面，那么这个我们在后续涉及到集群的时候，我们会着重的去说一下的。
