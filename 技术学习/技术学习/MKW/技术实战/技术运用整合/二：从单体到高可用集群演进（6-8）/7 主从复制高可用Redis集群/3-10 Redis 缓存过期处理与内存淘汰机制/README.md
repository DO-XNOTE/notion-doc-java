其实如果说我们的缓存就是说你是可以设置一个 expire 的一个过期时间，一旦过期了，那么应该怎么去处理呢？对于 Redis 来讲的话，它有这两种的方式，这两种方案可以去处理。那么这个是针对那些设置过 EX 派时间的一些key，然后这个 EX 派的时间已经是过了，但是这个时间一旦过了以后，这个 key 的话，它虽然是查询不了了，但是其实它还是会占用我们服务器的一个内存，所以它就会有一种策略，一个叫做主动删除，或者说是定期删除。另外一个叫做被动删除，它也称之为是惰性删除。
