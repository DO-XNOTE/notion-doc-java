hello 我们课网的各位同学们，大家好，咱从这一章节开始，就要揭开整个微服务大篇章的序幕了，相信此刻同学们的心情一定是比较非常激动的很，那咱还等什么，我们只出好戏就要上演了。欢迎大家收看史诗级红篇巨著微服务架构连续剧的第一季微服务初体验。在这一季当中，主要跟大家介绍一下微服务架构的理念，让大家对微服有一个比较直观的认识。在开始之前，我先跟大家介绍一下整个微服务课程系列，它的课程内容都有哪些？咱微服务阶段的课程大概安排了两个月的学习时间。那咱先来看一下未来两月咱都有哪些行程安排。咱在第一个章节里，先跟大家简单的介绍微服务的概念。我这里把它称作微服务三联，哪三联呢？第一个我要跟大家介绍一个 what 就是微服务究竟是什么东西？那接下来是一个问题叫 Y 也就是说为什么我们的应用要做微服务化，最后跟大家介绍 how 也就是说如果要做微服务化改造，那么都有哪些服务拆分的原则？那咱对微服务有了了解之后，我们就要引出咱这一个章节当中的主角 spring cloud 在第二章节的课程中，我们将跟大家简单介绍 spring cloud 的整体架构以及它的核心组件。那微服务三连加上 spring cloud 的简单介绍，这就是咱开始正课之前的准备工作了。


接下来画风一变，我就要跟大家展现真正的技术了。接下来我们就去看看在实战环节当中都有哪些新技术等着我们。那我们就有请微服务代表团的选手们登场，eureka gateway stream 还 strikes loose fin double config sentinel bus ribbon 这场面真的是眼花缭乱，目不暇接咱们码农不都是自嘲自己叫什么搬砖的对不对？这里面的每一个砖头块都是要让大家亲手去搬的，每块砖头都是一个独立的章节，同时他们也是构成我们微服务架构体系的核心技术组件。那我们何不就借这个机会给每一个组件打一个亲切的照面，先混个脸熟再说。那我们就从最左上角的那个叫尤瑞卡的老大哥带头大哥先开始。


OK 目光聚焦在尤瑞卡，这是我们 spring cloud 当中最核心的一个组件，它专门用在服务治理领域，大显身手，摄影师老师移动镜头。接下来目光聚焦在瑞本，他是 sprint cloud 当中负责负载均衡的一个组件，瑞本可以和其他的 sprint cloud 组件做无缝的集成。


接下来是一个重要角色 high streaks 它专门用在服务容错领域，咱平时听到过的降级、熔断等等高大上的名词，就由 high streaks 来搞定。接下来是 fin 它是用来简化服务间的调用，让我们调用远程方法，就像调用一个本地接口一样的简单方便。


在它之后是 config 它是分布式的配置中心，用来管理各个应用中五花八门的配置项，还可以在运行期刷新配置属性的值。 config 还有一个好搭档，那就是 buzzbuzz 呢是我们的消息总线，通常呢和 config 组件搭配使用，用来批量推送配置项的刷新。接下来呢是微服务领域中一个非常重要的组件 gateway 它是服务网关，根据一系列的路由规则，将用户的请求导向正确的后台。微服务。接下来就到了 slenth ，他是调用链追踪的专家，专门在日志 log 当中进行打标操作，串联整个调用链路中的上下游服务。紧挨着斯鲁斯的是 stream 组件，它是用来和底层消息中间件进行集成的消息驱动组件。海上最后一个组件是 sentinel 他的专业是限流。


OK 咱已经挨个认识了每个组件，接下来大家排排坐，准备照集体照了出场。好，这所有的组件当中，我们给它做一个分门别类，在这个过程中还有新的同学要加入。 OK 那全题都有。听好口号列队，咱看到所有的内容都被分成了上半段和下半段。其中上半段是咱 spring cloud 当中将要学习的所有组件技术。下半段是我们本章微服务课程中的附加内容。可以看到咱有两个新的同学加入，那我们也来挨个认识一下。


第一个部分是架构思考，我们将在微复阶段所有章节的课程当中穿插架构思考的内容，这里面包括技术选型、高可用的改造，以及我在过往经历当中做系统架构方面的一些思考和总结。除了架构思考部分，我们还要引入一些业界的案例。在咱的课程当中，我引入了大量的一线互联网公司的实际案例，大部分是我在阿里做业务系统时积累下来的所思所想、所见和所感。这部分是用来丰富同学们的眼界，让大家也能了解当前国内一线互联网公司当中他们的业务系统是怎么搭建的。


这些电商场景背后都有什么故事？ OK 最后一个内容就是大名鼎鼎的 double   double 并不是 spring cloud 当中的一部分，不过它却是实现服务治理的另一条路线。那就是 RPC 框架，它在国内互联网公司中的使用还是蛮广泛的，很多公司的服务治理框架都是构建在 double 之上。


OK 那这里就是微服务阶段所有章节的全貌。我们的整体风格是以实战为主，要让同学们能打仗，而且能打胜仗。当然了光能打仗做一个好士兵还不够，我们还要让同学们能做一个好将军、好司令官。所以我也会从架构思考和业界案例这两个方面入手，丰富微服务阶段的课程内容。那希望同学们在未来的两个月可以和我一起度过一段快乐的学习时光，相信我们会幸福的同学们。那我们下一节课程再见。
