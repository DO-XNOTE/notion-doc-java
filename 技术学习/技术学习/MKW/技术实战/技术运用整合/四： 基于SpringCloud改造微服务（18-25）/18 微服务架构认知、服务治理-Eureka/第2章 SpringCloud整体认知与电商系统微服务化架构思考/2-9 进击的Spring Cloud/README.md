这就像倚天屠龙记里面，张无忌在武当山跟张三丰学习太极一样，学到最后，张老头问无忌，孩儿你还记得多少？那张无忌说已经全部忘掉了，这就是我们常说的把所学的知识全都还给了老师这样一个说法。那前面的每一个章节中我们都学习了大量的新技术新知识，那也经过了很长的时间，短则一两月，长则三四月。如果再问同学们，前面章节学到的某些具体的技术细节可能已经忘的差不多了，什么都不记得了。咦我在哪？我做了什么？我是谁？所以说时间是抚平一切记忆和伤痛的一味良药。

因为 Redis 后面分布式章节的知识有很多细碎零散的知识点。如果同学们在平时工作的过程中，没有机会把这些技术用于实际的项目，那是非常容易遗忘的。那我们很多的知识点都集中在分布式章节的 Redis 和 ES 之间。所以我们为了帮同学们加深对技术的理解，采用 Redis 做改造，然后让同学们动手，把 Redis 之后的技术从头到尾的再集成到项目当中。
