这里的 field 会稍微多一些，它跟什么有关？一个是时间。另外一个就是跟服务续约有关系。好。第一个， column 是 current time current time 就是当前系统的时间，这个时间并不会刷新，只有当你主动的刷新整个页面的时候，它这里才会刷新到新的时间。
那第二个， column 是 up timeup time 并不是指分钟的意思，因为这看上去有点像 1 分 30 秒，对不对？它实际上是一小时 30 分钟。它的含义是指自从这个注册中心启动到运行至今所经过的时间，那接下来的一个字段就比较有意思了。


Expiration enable. 那它是什么含义呢？它是否启用租约过期，它跟服务续约、服务剔除、服务自保三个功能都有一些关系。那假设当前我的服务自保处于关闭状态，也就是说如果你的租约过期了，你没有及时给服务续约，你作业过期之后，我的注册中心是可以利用服务剔除把你的服务下线掉的。因此在这种情况下这个值默认是 true 同样的，我们这里看到它是 false 对不对？那说明什么？说明我们目前自我保护自保机关是开着的。
