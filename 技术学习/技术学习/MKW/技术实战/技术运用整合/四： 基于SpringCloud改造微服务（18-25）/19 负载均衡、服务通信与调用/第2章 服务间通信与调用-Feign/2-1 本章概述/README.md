我们对服务间的调用做一个简单的介绍，主要说明白两个问题，第一个问题就是啥是愤。第二个问题他干了啥？好。第二部分我们来深入 fin 这里面包含四个部分的知识。首先是分的体系架构。接下来我们看它底层是如何运作的，它底层的机制是什么？往后我们来关注一下它的动态代理是怎么实现的。大家知道动态代理不仅是在 filin 里面，它在 spring 里面也是一个非常非常重要的知识点。对不对？面试中经常出现的高频问题，那这里我们就一并给它了结掉，说明白说清楚大家以后面试再也不怕碰到这个问题了。
