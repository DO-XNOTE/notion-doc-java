如果说我们在使用数据库做一个截止的话，其实我们再去搜索字幕库网或者说是在搜索用户的一些内容的话，其实我们是可以在多个列之中，在多个字段里面我们都可以去做一个查询。比如说我们现在在这个里面来看一下，这边有一个 nike name 是慕课网对吧，然后这边 nike name 里面是有一个木这个字。那么现在当我们要去搜索的话，我们能不能再去把这个 nike name 或者说其他的一些列字段都能够加入到我们的这个搜索里面去的，那么其实也是可以的。要去加的话其实很简单，在这个后方我们加上一个逗号，然后再去加有一个叫做 fuse 这个其实就是他查询的那些列，你可以去加上一个数组，在这个数组里面去加一些内容。比方说现在我们要去查的话，一个是 description 另外可以去加上一个叫做 nick name 我们可以通过这种方式去查。那么这种方式其实就是我们可以在多个字段里面去查。当然我们语法肯定是不对了，因为我们要使用这种方式去查的话，这个 match 它其实是单个去查的，我们要把它做一个修改，使用多重查询叫做 multmatch multi match 使用这个去做对应的搜索。


搜索的话，在这边我们慕课网这个时候它就不再是一个迪斯科品，它是一个科瑞，它是一个查询项，是一个内容，外加它的一个 files 相当于是把它的一个字段和它的一个对应的属性给分割开来了。那么这样子你就可以去做一个相应的搜索。原先我们搜索出来是有几条记录的，有四条记录。那么在这边我们可以这样子去点搜索之后，你会发现在这边多了一条记录。
