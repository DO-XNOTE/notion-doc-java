下面咱们来看一下这 64 匹特都是由哪些组成。首先第一位是一个0，因为它是一个二进制的数，一共 64 位。第一位是0，那说明它是一个正数。这第一位 0 也是，写死的不能是一对吧？如果第一位是 1 的话，那么这一串数就变成了负数，第一位是0，这个固定不变。然后跟着是 41 位的时间戳。这个 41 位的时间戳它记录的是一段时间。这一段时间是什么呢？是你当前的这个时间减去你设置的这个开始时间，这两个之间的毫秒数，就是这个 41 位。这个开始时间是雪花算法，内部可以设置了这 41 位的时间戳，可以能维持 69 年，这个时间也是比较长的，大家可以放心大胆地去使用。然后后边跟的是 5 位的机房 ID 这个它的官方写的是 data center ID 是吧，就是数据中心的那个 ID 然后再跟着 5 位机器的 ID 有的这个雪花算法的生成器，会把这两个给它合并到一起，统一的写成一个 10 位的这么一个机器的 ID 它分成两个 5 位，也没有多大影响对吧，主要是标志着你当前的这台机器的唯一的一个序列和前面的这个时间戳去给它分开。
