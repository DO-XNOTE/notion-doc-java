然后这两个分片库都统一的从 ID 序列生成器当中去获取 ID 对吧？比如说这个分片库一是 T order 1，分片库二是 T order 2 是吧。之前重复，也就是说咱们设置的这个 ID 序列都在每一个分片库当中自己去执行，它们之间并没有联系。那么这个时候就有可能 ID 重复。
现在咱们统一的从一个集中的地方从这个 ID 序列里边去获取 ID 就可以保证它不重复了对吧？咱们这个 it 序列生成器在咱们的示例当中，买 Kite 是支持的沙晶类 abc 不支持这种 it 序列生成器的方式。这个那一会儿咱们就给大家演示一下在 my cat 里边如何配置这个 ID 序列生成器。
