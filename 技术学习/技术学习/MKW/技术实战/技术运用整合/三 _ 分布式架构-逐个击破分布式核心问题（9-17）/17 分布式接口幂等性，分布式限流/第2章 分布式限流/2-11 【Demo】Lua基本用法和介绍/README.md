它首先作为一个轻巧的脚本语言，有与其他脚本语言共同的一个特点，就是短小精悍。待会儿同学们写入外程序的时候就可以体现到它的这种简洁性了。咱往小了说它是脚本语言，往大了说也可以说它是一个编程语言。不过 Lua 相对于 Java 这类面向对象编程语言，它的入行门槛非常非常低，咱不说 30 分钟学通，至少学个两三天，大家应该都可以动手的写一些简单的 Lua 脚本程序了。
那 Lua 脚本都有哪些具体的应用场景呢？我们这里给出了两个比较常用的，一个是嵌入式的开发，第二个是插件开发。 Lua 脚本可以作为一个小的代码片段嵌入到其他的开发语言或者平台当中。比如说它可以为某款特别著名的游戏编写插件，是什么呢？为了部落。对了，魔兽世界。


当然了，除了插件开发以外， Lua 其实也是可以上得了台面的。它也可以用来作为主打编程语言写一个不错的小游戏。比如说愤怒的小鸟，大家瞬间感觉哇撸好厉害。对不对还有更厉害的，它可以完美继承 Redis 什么叫完美集成呢？它的完美体现在以下的这几个方面。第一，从 Redis 的角度来说，它内置了一个 Lua 解释器，大家没想到吧， Redis 咱平常用起来都是把它当做缓存来用的。谁曾料想，它里面还内置了一个 Lua 的解释器。并且 Redis 在执行 Lua 脚本的时候，是天生的原子性操作，还没完， Redis 还可以支持预编译的 Lua 也就是说我把 Lua 脚本传入到 Redis 进行一个编译。完了以后，我下次调用的时候直接调用编译好的 Lua 脚本就可以了，不用再次上传。那这里面具体的细节，我在后面的章节再跟大家一一描述。
