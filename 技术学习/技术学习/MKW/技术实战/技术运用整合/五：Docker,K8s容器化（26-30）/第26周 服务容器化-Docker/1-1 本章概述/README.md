相信大家在上一讲姚半青老师的那个内容里面已经听到了很多关于微服务的改造，但是到底改造完以后怎么样部署到我们实际的环境当中，使它能够完成高可用、安全性、稳定性、可扩展性这些需求，那我们这一讲就会带领大家通过一个容器化的总体介绍，再逐步到后面几讲，包含倒渴可奥、方锥、mesos马阿松，还有 kubernetes 以及最后的弹性扩缩容章节，实现把应用完整的部署到我们整个生产环境的这样一个介绍过程。


那也希望通过这一讲以及后面的几讲，带领大家从一个架构师的角度去看，我们在实际的这个开发过程当中，或者说部署过程当中出现什么样的问题，怎么样进行分析筹谋和应对。那同时我们在这一讲更偏重于总体框架的介绍，我们会希望横向比较，比如像我们的那个卡尔方瑞和德克两个热门的容器技术，以及像库布内提斯马拉松，还有梅瑟斯这些主流的编排技术，他们的优核列同时也希望带领大家从虚拟化开始到逐步出现容器化到容器化蒸蒸日上，以及对于未来的一些考量。
