是首先要把我们的后台准备好。那上一节其实已经准备了 MySQL 的，也准备了 Redis 基于我们阿里云的这个服务。除了这个以外，我们还有哪些服务没有准备好呢？其实是需要一台计算节点，这个节点用来生成 Doc image 做一些中转处理。我们如果自己的笔记本电脑是台 Linux 的，那基本上就已经可以胜任这样的功能了。那如果是一台Mac ，大家对 Mac 的这种安软机安装很熟也可以。那我这里也是租用了阿里云的一台 Linux 机器三投 S 机器来做什么计算节点，来做一些业务中转来做一些容器的那种准备。
