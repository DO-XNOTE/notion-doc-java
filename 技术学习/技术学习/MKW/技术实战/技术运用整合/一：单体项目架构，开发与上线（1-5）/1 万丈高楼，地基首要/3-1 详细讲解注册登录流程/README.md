实现分类其实是电商网站里面比较重要的一个内容。分类其实就是商品的类别。商品的类别其实又会分为有很多，有大类、小类、子类等等。**对于我们来讲，我们只需要通过一张数据库表就能去实现，并且是实现一种无限递归的形式**。说下一个是首页的商品推荐，这个其实我们在之前的一个演示里面其实也给大家演示过，在我们使用鼠标上下进行滑动的时候，每一种分类的一些商品其实都会在我们的页面上去进行一个加载，这个我们在首页里面不会一次性的把所有的数据都拉取出来，是根据页面的一个滚动逐步把数据去拉取出来的。这个其实是一种懒加载的模式，这个可以提高我们用户的一个使用体验，也可以让用户在初次进入到首页的时候，页面可以完整的加载出来。
