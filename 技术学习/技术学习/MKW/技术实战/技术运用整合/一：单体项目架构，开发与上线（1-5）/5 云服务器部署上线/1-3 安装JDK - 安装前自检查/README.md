随后回到咱们的文档，在我们要去安装一个 JDK 之前，我们要去先确保在我们的服务器里面没有安装默认的一个JDK。如何去检查？通过 Java 杠 version，那么这个是在 windows 平台或者 Mico s 上，只要我们安装好一个 JDK 之后，我们都会通过这个命令去检查，所以我们直接拷贝在我们的这个命令行去敲一下，这个时候会发现我们当前环境里面没有JDK，但是如果说你是在**本地虚拟机里面去进行安装的话，那么要去注意默认的Centos。不管是7点叉还是 6 点叉，那么它都会为你去安装一个 open jdk。这个你是需要去卸载之后的，你要去卸载之后你才可以去安装新的JDK，这一点你是需要去注意的。**
