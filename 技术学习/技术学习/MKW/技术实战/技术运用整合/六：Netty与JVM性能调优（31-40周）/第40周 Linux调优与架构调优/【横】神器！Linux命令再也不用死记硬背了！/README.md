好，说了这么多， tldr 该如何安装？首选的方式有两种，我们来看一下 tldr 的GitHub，往下翻，在这里可以看到你可以使用 npm 或者 PIPE3 去安装，前者需要先安装 note Jesse 环境，后者需要先安装拍摄环境。在安装的时候， TLDR 需要访问 GitHub 并下载相关的资源文件缓存到本地。如果你的 GitHub 无法访问，可以看一下我的上一个视频里面手把手的教你如何流畅地访问GitHub。除此以外，社区还提供了数十种安装方式，在 wiki 文档里面支持这么多种的安装方式，如果你不想安装也无所谓。他们还提供了 PDF 版本，可以点击下载PDF，然后想查某个命令的话，只要到 PDF 里面搜索就 OK 了，这就相当于是一个命令速查指南，对吧？OK，这节视频就到这里，希望能够给大家带来帮助，关注我分享更多实用知识。
