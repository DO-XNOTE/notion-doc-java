在这一章，我们首先会通过各种实战，从而帮助大家积累调优的经验。这里面的大多数实战都是弹幕在实际项目中遇到的，希望同学们能够有所收获。

第二，这一章会使用到理论篇的理论去支撑我们实战，使用工具变的工具去帮助我们快速定位问题。在这一章，我们会把前面的内容糅合到一起，从而帮助大家真正的融会贯通。


当然了，这一章的内容相对会综合一点，因为经常会牵扯到前面的内容，同学们可以拿前面相关的章节去相互印证。如果遇到某个问题你不能确定，也建议回过头来复习一下。另外，一定要多动手多操作，这样才能夯实基础，为了在调优的路上走得更远。


另外要说明的是，咱们的实战篇也不是全程实操的，里面也包括了一些理论知识的增补。同学们可能会觉得这样安排有点怪异，为什么不直接在理论篇讲完？这是因为这一章里面涉及到的理论放在理论篇会比较抽象，而且比较枯燥，而放到实战片里面去探讨，理解起来相对会轻松一些。


另外，这一张也包含了几款工具篇没有提到，但是也非常好用的工具。之所以不在工具篇一步到位，主要是因为工具篇都是围绕 JVM 的相关工具去展开的，而这一章的工具并不局限于jvm 相关的工具，而是面向实战的。做实战的过程中，哪个工具比较方便，我们就用哪个工具。


第三，在这章里面包括了大量的技巧和经验总结，希望能够启发同学们的思路，对同学们有所帮助。
