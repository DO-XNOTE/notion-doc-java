直接内存是一块由操作系统直接管理的内存，也叫对外内存，可以使用 unsafely 或者是 bytebuffer 来分配直接内存。另外可以使用 Max direct memory size 去控制直接内存，默认是0，表示不限制。


好，现在我们已经知道什么是直接内存了，大家想对象一般不都是分配在对内存的吗？为什么还要再弄一个直接内存？
