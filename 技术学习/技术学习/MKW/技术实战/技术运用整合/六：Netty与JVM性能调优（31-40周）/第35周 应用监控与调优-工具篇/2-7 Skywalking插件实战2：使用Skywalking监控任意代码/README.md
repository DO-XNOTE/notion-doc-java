我们该怎么监控这个类呢？ a p m customize enhance plugin 这个插件能够满足我们的需求，它可以用来监控任意类的任意方法。使用这款插件细节还是比较多的，所以我专门写了一篇手记，一起来看一下。这是一篇手把手的手记，下面我们就来对照手机操作一下。第一步，移动价包。这是一个可选插件，所以需要把它移动到 plugin 字目录。接着看。第二步，编写增强规则。这是一段 x m l，它用来描述监控哪些类的哪些方法。我们创建一下叉 MA 右击创建文件，文件名称随意，我们叫enhance。点Xmail。文件的路径也可以随便。我们就把它放到 agent 目录吧。存储。创建完叉票之后，我们用 idea 打开。我们不妨 copy 这里的 XML 作为模板，粘贴叉妙里面每一段表达什么意思？我在手机里面也有详细的描述。首先 class name 表示要被增强的类，通俗的说就是想要监控的累。对于我们 page helper 右击 copy reference 获得包名类名全路径。粘贴， method 表示类的拦截器方法，通俗的说想要监控的方法。
