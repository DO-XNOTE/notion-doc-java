首先我们来配置一下国内源，这一步只是为了提升安装的速度，你也可以不做。先把源配置文件备份一下， copy 粘贴。接着使用中国大提供的圆 copy 粘贴，使用 apt get update 更新一下。


更新好了。最后使用 wget 下载一下 Percona  release 仓库的安装包粘贴。他说没有 wget meaning，这是因为这台机器是一个最简安装的Linux。我们使用 apt get install wget what ？安装好了，再次执行 wget 下载 Percona release。下载好了之后，使用 

dpkg 安装 Percona Toolkit release。


报了个错，他说 Percona Toolkit release 是依赖了 LSB release 的，然而 LSB release 没有安装，因此我们需要 apt get install 一下 lsb release。他就说 RSB release 依赖了 Python 3HISQ。建议我们使用 abd fix broken install 谁，可以发现 Procona release 已经安装完成了？


做完 Percona Toolkit release 之后就可以安装 Percona Toolkit 了。 copy 粘贴，安装好了。安装完成之后可以使用 pt 按 type，会自动提示相关的命令，每一个命令就是一个使用工具。这是 Percona Toolkit 的安装。下面来挑选这款流行的工具演示一下。
