好。第二个比较重要的字段是time， time 表示线程在这个状态下面持续了多久。第三个比较重要的字段是state，它表示这个线程正在执行的操作事件或者是状态 state 的取值非常的多，总共有一两百个。同学们可以对照这几篇手记去看一下。好。
