希望同学们经过学习至少要达到这样的水平。首先，要能够知道这些工具大致上的功能。比方提到建map，就知道这是操作堆内存的，提到建stack，知道这是栈相关的一个工具。第二，在今后做相关调优或者是故障排查的时候，要能够想到去使用合适的工具。


第三，希望同学们能够对照视频，把课上用到的工具都去实操一遍。有的章节你可能看老师操作起来非常的轻松，但其实有很多的细节只有你亲自操作了才会理解到。如果达到这 3 点，本章的教学目的就算是达到了。至于这些工具，具体的参数，大部不建议同学们去死记硬背。特别是命令行工具，同学们只要能够读懂对应命令的帮助文档，也就是对应命令的杠 help 所展示出来的信息，能够看懂并且会使用就 OK 了。


当然了，如果你有余力背诵一部分参数，也是不错的一个选择，至少能够帮助你在今后使用这些工具的时候，提升一定的效率，对吧。
另外，大漠针对这一章专门编写了一整套的手机，也希望这套手机能够在文档完备性以及实操的角度给同学们一个参考。这里大部已经把这套手机编译成了一个 PDF 文件，一共有 70 多页，同学们感兴趣可以下载看看。地址在这里，这节课就到这里，谢谢大家。
