主要帮助大家梳理 j v m 调优相关的理论知识，从而构建起调优相关的知识体系。第二是工具篇，主要普及业界非常好用的调优工具，让你今后在调优 j v m 或者是排查 j v m 相关问题的时候，能够如虎添翼，不再刀耕火种。


介绍完工具之后，我们会过渡到实战篇。在实战篇里面，大漠会结合自身的工作经历，带大家手把手的做几个 GFM 调优实战。在这个过程中，会和大家一起分析调优的思路，并去手把手的操作调优，帮助大家积累调优经验。下面我们开始进入第一部分理论篇的学习。在这个部分主要会探讨以下知识点。首先是 j v m 的内存结构，第二探讨类的加载机制，接着会探讨 Java 编译器的优化机制，比方法、内联站上分配、标浪替换等等。再之后会探讨 Java 里面的垃圾收集算法以及垃圾收集器。


总的来说，理论篇的知识比较的琐碎，而且细节非常的多，理解起来也有一定的难度。因此建议同学们观看这一章的时候可以找一个比较舒服的姿势。另外一定要边看边记，否则过短时间就会忘记了。要知道好记性不如烂笔头对吧？下面我们一起来探讨 j v m 相关的理论知识。
