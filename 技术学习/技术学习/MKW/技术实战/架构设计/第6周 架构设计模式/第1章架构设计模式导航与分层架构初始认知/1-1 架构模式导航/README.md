我们也有各种各样的案例来帮助大家能够迅速地成长和掌握了解这些架构模式和架构的设计方法。同时课程内容大概就是分成这样几种，我们大概有四种架构模式的分享，有两种典型的架构设计方式方法分享给大家，同时我们还有很多的经典架构的应用，通过对它的这各种解析和剖析的脉络，了解和掌握更多的价格模式的思想和方法。通过例子也罢，还是通过实战的代码或者设计图来帮助大家了解价格模式的各种运用方法。
