# 1-2 分层架构前菜

前面说了这么多，下面我们就开始正式学习第一种架构模式，这种架构模式是一种非常古老的智慧，在我们中国古代的经典著作西汉的杨雄手写的太玄军中，就对我们天地的各种层次划分。大家可以从我们现在屏幕上所出现的这九天的电影，大致的了解一下古代人对天地间的一个朴素认知。

此外，我们中国的口语中经常还说什么九霄云外，其实都差不多一个意思，就是古代人对世界的认知把天地划分了很多层次，每个层次各司即止，你要从一个层次到另外一个层次几乎是不可能，因为他们彼此是隔绝的，同时要跨层次交流那更是难上加难。所以这么一说，是不是和我们的平时生活中有很多这样的例子，为了某些目的，我们将事物划分成不同的逻辑层次或者物理层次，这也就是我们这个章节所有学习的架构模式。分层架构相对于其他的架构模式，

这是我们很多人在工作中已经非常熟悉的一种架构了，可能大家的实践经验都已经有了，那么在实践之外，我们是不是也要学习一下背后的理论知识和抽象过程，因为大家都知道架构和模式都是由实践中抽象而来。
