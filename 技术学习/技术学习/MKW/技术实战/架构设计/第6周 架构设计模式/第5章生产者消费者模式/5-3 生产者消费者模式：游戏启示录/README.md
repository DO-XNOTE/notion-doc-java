就相当于你作为一个司令官，或者你作为一个帝国的一个操控者，然后你要组织国家的生产，同时也要产生军队和对方进行对战。那这些经典的当然也比较包容年龄的，包括对这个帝国时代新纪念霸，包括后面的各种各样的这种即时战略游戏，都是非常非常流行的当时，但这个也不要放了绵阳我这个地方举的例子是用帝国时代举的例子，其实玩这种即时战略游戏，如果大家有经验的话，你会发现你作为这个最高决策者，其实你就一直在想一个问题，我应该多放点人去生产还是多报一点兵？那所以说打法其实有两种，一点农夫流就通过庞大的这个经济体系，或者良好的这个经济体系来拖垮对手，包括兵就没有那么多。那另外一种就是穷兵黩武，疯狂招兵，直接爆兵把他给抱死。这两者反正在市场上或者当时进的一些比赛当中都有，所以你看在这个地方，你就决定着是分配了多少资源到不同的这个上面来制造东西。


哎，就是什么造农民，我们多造点农民，让农民来多生产来，这样子我的经济很好，即使你打我一下，我还有个分基地，或者我在别的有很多，不管是食物、金钱还是材料我都很丰富，我们马上可以开分基地，同时我再发起反攻。另外一种就是疯狂刨冰，直接把你给灭了，只要在你的个经济成长性没到一定程度的时候，这个曲线总归有一个交汇点，那我刨冰的数量大于你生产的数量，我砍掉你一个建筑，毁掉你一个兵团，杀掉你一些农民，而你后续造的没有这个快，那我也会赢。诶，所以说这两者大家就可以想象一下，一个抱农夫的这是不是增加经济实力，同时消耗经济实力去造出来的冰，那就是一个消费者，前面是一个智者，后面是一个消费者，那如何在这二者之间取得一个平衡？你作为这个游戏的操控者，就是你时时刻刻你脑子里面的一根弦，但当然不是说时时刻刻你都是最高这个优先级，但是在特别是发展的前期，你这两者的路线差别非常非常大的。


之前我们玩这个游戏的一个什么农民流程，直接从很原始的状态，因为农民就开始打仗了，就这里后面就是进化出各种各样的战术，这是基于这个什么升级层级，一步一步升级出来的。那穷兵读过兴趣的时候，我很早就开始报兵，另外一种就是我一开始轻轻地防御你一下，把经济搞起来，把这把你搞死，哎，所以这就是一种生产者与消费者模式的核心问题，你怎么解决这二者之间的平衡性？在输了怎么来控制到底是我多一点，农民好，发展的经济快一点，为后面打下雄厚的基础，还是我多抱了点兵，直接把你给吹垮了？其实这个策略可运用的一切的即时战略游戏当中，甚至是那种微软的那个文明不知道要玩什么。


玩过一样的道理，你愿意投资多少升级的科技术，还是愿意去穷病毒发展的？一个国家的文明程度还是很大差异的，反正这种就是深圳的消费者的一种迷思。那另外一个就是咱们中国小学生的事情难题，

因为只要在中国读个小学的人都应该知道这个题。我的一个池子有一个水龙头不停地进水，另外一个水龙不停地放水，那到底这个时候什么时候池子被盛满，什么时候这个池子会放空它一般都是这样一个套路，这个题目就是只要在中国接受过教育的人应该都是知道的。老大，其实这个你就可以看进去的。


水龙头是不就是生产者？那初始的水龙头是不是就为消耗者？二者之间这个差数是怎么样的？谁多谁少？最后这个池子的结果最早一杯注尽，但是其实这就是一个很原始而且相对简单的这样一个，不像前面说造农夫还是造兵的这么难的学。这个，因为这个是相对比较线性的，它不是曾经的指数性变化，因为这个水格的速度如果是恒定的话，它相对来说比较简单。这个地方其实就是你思考一下，这个其实也是一个生产者，一个消费者。后面大家如果看到我们后面的这个讲解的理由体系的，你会发现跟咱们这个池水这个例子和前面这个刨冰的例子都是非常非常接近的，总归是有一个地方在生产，有一个地方在容纳，有个地方再流出，就是这样子一个理论体系，基本上这个模式就是这样子的。后面在理论体系当中它就可以看到一个更详细的，我们先在这里埋下一个伏笔。
