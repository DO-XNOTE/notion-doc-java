好，那这个时候在数据中心里面可以听很多的内容，那当数据中心跟其他的中心进行交互的时候，通常可以这样有一个网络连接，那通过这样的网络隔离，再在隔离的过程当中还有一部分的网络连接，我们通常都是这样，是吧？我们数据中心是既和外网有联通又有隔离，不是任何人都能访问，所以这根黑视线表示很多内容会被隔离，但部分应用可以访问。
