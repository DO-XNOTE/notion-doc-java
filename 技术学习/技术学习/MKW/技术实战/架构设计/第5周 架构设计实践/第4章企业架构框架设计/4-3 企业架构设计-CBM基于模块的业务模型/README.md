那这个模块其实就是组件了，那它的模型就是业务模型，所以也可以简称叫模块组件、业务模型。那这样的一个模型如何去跟前面的 SCN 来贯穿的？那我们首先要了解一下业界最流行的 CBN 到底长什么样子，然后再看一看 SCN 和 CBN 如何来进行贯穿和互通。那重点先要了解一下大家说的 CBN 是一个什么样的核心思路。 CBN 本质上是一个表格，它有横列，那它的这个列重点来看一个企业应该如何把这个企业的所有的业务，还有他所有的部门进行分组，那这个分组就是 CBM 的列，待会我们会有一个实际的例子来给大家展现。
