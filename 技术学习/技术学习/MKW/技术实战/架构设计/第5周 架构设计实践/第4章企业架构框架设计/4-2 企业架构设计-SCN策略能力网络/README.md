大家就会想到华为，那狼有很多特性，那华为到底是什么样的一个整体企业战略？什么样的方法论？那我这里以它的全球化战略为例，你可以查阅相关资料才能找到它在全球化上有这样几个方向，一个是营销，它营销采取什么发展中国家为先的外交低价的策略，逐步产生什么发展中国家包围发达国家的全球化战略，这很像狼的什么什么文化？狼通常是先啃，对吧？容易啃的骨头是吧？他不去啃大象了，一发现都啃小鹿，然后再聚集起足够多的狼再进攻大象。


我们狼性文化里就是先从薄弱点开始，先从容易切入的发展中国家开始，跟他们形成一个配合，帮助他们进行通讯系统的搭建，然后再逐步把我们的品牌打出去，成为全球化的一个品牌。那除此以外，创新也是什么？华为全球化战略的重要部分，它在中、英、法、日、俄、美等成立了 18 家研发机构， 36 家以上的这个创新工场。通过这种方式什么？有没有像狼群的这种群体行动？狼会包围一个动物，会从各个角度，对吧？各个方向去包围它，然后进行总攻，那同样的道理是吧？华为在包围全球战略上，在各大的主要的研发机构，或者说各大主要的这个人才聚集的地方，搭建自己的研发中心，搭建自己的创新工场，然后形成包围全球的一个态势，那除此以外本土化也是很关键的一个战略。


那本土是指什么？在华尔委销售的各个地方，比如像中东，比如像非洲这些国家，它都采用本土雇佣一些我们的技术人才。那同时采用一些本土的产品进行配合，说，甚至于开发一些产品专门给本土使用，同时在本土也会成为纳税大户。那华为在中国是纳税大户，在什么非洲，在中东，在各个小国家也都是当地的纳税大户。这就有点像什么？这有点像狼群的那种，就是链家的感觉，就是每一个狼孤狼受伤以后，他就会回到自己的家里去养伤。


狼对领域是吧？对于这个地域的感觉很浓，那同样道理，我们每一个什么，我们的这个华为的这个分部，它都有很强地域性，它会利用地域的人才在当地的产品配合，然后成为当地的一个知名的企业，通过这种方式能辐射当地，对吧？帮助这个华为打开当地的市场。那这样三大战略我们会看一下怎么样映射到it？那首先要在了解一个企业的战略，特别是它的什么，它的企业整体的战略和整体的价值，然后采取什么映射到我们的it，那这就是 scn ，对吧？**策略能力模型要聊的就是你要去为一个 it 企业，或者是一个任何的互联网企业，或者是一个传统行业企业去制作 it 解决方案的时候，先了解它的战略，**我们可以再举一个例子，看看我们的快速消费品市场的这个全球的排行的，这个前几的这个大公司宝洁，对吧？它的多品牌企业策略。
