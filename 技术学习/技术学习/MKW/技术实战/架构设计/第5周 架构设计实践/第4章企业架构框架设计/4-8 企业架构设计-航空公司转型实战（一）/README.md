这家公司被很多的低廉航空所冲击，低廉航空现在不光是票子卖的便宜，还提供少量的互联网化的销售业务，包含一些互联网化的一些直播点播平台的推送，很多的这种景点，很多的时尚明星都跟 DNA 航空产生了一些交互，对整个航空市场，对整个的这个我们大兴集团进行了很大的冲击，如何应对这种情况？那第二点就是在用户不断追求低价，不断追求高追加服务的情况下，用户的品牌粘性、忠诚度越来越低了。
基本上都是什么找一些这个比价软件，通过一些各种各样的搜索引擎来找我们去某一个景点或者某一个城市最便宜，同时服务相对比较好的航班来并行比价和购买。那这种情况下怎样保持我们航空公司大型集团的优势，提高品牌的粘度，使得用户能够忠诚于这个品牌，这是一个难点。


最后一点就是什么这家公司通过大量的收购和合并，已经有了很多子公司和分公司，同时在全球有一个很多的业务，但是这些子公司、分公司和全球业务并没有进行 it 的整合，也就是有很多 it 系统来独立进行结算和处理，甚至于整个航空集团的积分都没有进行完整的对接。那这种情况下用户的体验，整个市场的这个划分都有问题，怎么样保持很好的市场覆盖，降低总体的运营成本？诶，这就是整个我们航空大集团转型的这几大难点。诶，是一个很大的项目。
