首先，你要了解你的架构设计的目标是什么，做到什么样的程度才是架构设计，才完成了架构设计。其次，我们要选择一个合适的方法，学得越好，我们架构设计的时间成本就会越快，我们投入的资产或者投入的这个人员就会越少。那最终我们要什么？不管你是选用什么样的目标，采用什么样的方法，必然会有架构设计输出，在你的规划环节就应该制定好我需要输出什么样的工件，工件在英文里面叫 artifact，中文就叫做工件。
