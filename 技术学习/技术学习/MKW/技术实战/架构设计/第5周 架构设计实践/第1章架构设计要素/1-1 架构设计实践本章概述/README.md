那这节课首先会重点强调一下什么是架构设计，架构设计的输入跟输出是哪些，如何来收集输入，如何来准备输出。那其次会聊一聊整个架构设计的核心方法论。架构立方体，我们这个章节会以架构师思维 architectural thinking 为主体，而它的核心思想架构立方体将围绕整节课程全程对大家进行展现，那通过架构立方体，我们会探究一下架构设计的具体的方法论和步骤是什么？那在整个过程当中有两大模块，一大模块是功能性的模型，它通常叫做组件模型，具体来看它会定义模块的内容，细化模块的细节以及映射模块，到可实现了一个一个的单元和组件。那第二大模型就是非功能性模型，我们也叫做运行模型，那这里会尝试把架构进行单元的分离，并且从不同的视角来进行架构的转换。
