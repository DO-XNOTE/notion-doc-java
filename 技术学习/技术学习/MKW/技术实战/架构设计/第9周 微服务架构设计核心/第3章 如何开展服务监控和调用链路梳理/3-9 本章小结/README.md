# 3-9 本章小结

微服务架构的系统需要搭配大量的支持性组件，链路追踪组件和埋点组件就是一个例子。在真实项 目中借助Sleuth之类的组件做日志打标，日志文件经过ELK或EFK做聚合并提供搜索功能。在线上故 障排查的时候，调用链梳理是很重要的一个环节，不仅可以迅速找到出问题的环节，还可以上下追 溯到前后调用的服务。


至于业务埋点，最廉价的手段就是通过Log组件写日志，但大多数一二线大厂都有专门用于做埋点 的中间件。比如阿里系的鹰眼系统，很多公司也偏向使用基于couchbase的埋点组件。
