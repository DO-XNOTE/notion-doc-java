我们首先**从这个可执行层面来看一下，比如说你在阿里是一个P8，那你到了中小公司，或者是说创业公司，完完全全可以是一个企业架构师去决胜千里，运筹帷幄。但是如果回到 BAT 这种体量的大公司里面做企业级架构师，基本上是不太现实的事情，因为它每一个部门，它每一个事业部体量都非常大，所以我们从可执行的层面来说，我们把这个企业级架构式的目标降低一些，我们把它弱化到事业群、事业部，再往下沉到更现实的层面**。
