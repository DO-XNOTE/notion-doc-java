在你的业务已经具备了一定规模的时候，我如何通过技术手段让你的业务成本迅速降低？那这里面其实简单的来说有两个层面，那一个是减少人工成本，那我这里举一个例子，比如说淘宝这样的平台，哎，你作为一个商家，我要上传图片商品主图，那你自然在后台要去做一些甄别，禁止你上黄图。
