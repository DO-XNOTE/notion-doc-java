这一章我们会尝试从一个横向的角度来看一看国内一些知名的云平台厂商，比如像腾讯、华为、百度AWS，当然还有前面什么反复实战反复提到的阿里云，它们的差异。而重点会挑选 4 个平台，也就是腾讯、华为、百度、AWS，跟大家在这个平台上面实战一下他们所特有的所引以为傲的一些技术栈。


这张之所以要放在最后，对也是作为一个什么 last but not least，对吧？最终也是最重要的一件事情来聊是因为什么合适的才是最好的。我们上云首先要考虑什么样的应用需要上云，我们需要有什么样的目标，我们需要走什么样的路径。真正上云的过程当中，其实很关键的一点就是学什么样的公有云平台，如何去打通公有云和什么自己的自建机房，实现混合云逐步转移到云平台这样一条路径。中间一个难点就是什么找到合适的云平台的供应伙伴。


到底谁才是最好的？辅媛老师总结出来就是合适的才是最好的是吧。我们美女有五花八门不同的这种风格的，怎么样才能找老婆呢？当然要挑跟自己最匹配最合适的。我们上云也一样，我们上云不光要看技术，要看平台，要看合作的意愿，同时也要看我们能不能从云平台上面得到我们所需要的支持和保障。对既有技术的也有什么，也有支持，也有服务。同时还要考虑到什么，自身到底有多少米，多少资金能够上，买多贵的资源。最后一点就是how？ how 里面我们会尝试什么，有理论，但是非常少，大部分就是实战，让大家真真切切感受到其他不同的云平台的实战的体验，了解到天上的云彩五花八门，五颜六色，非常精彩。
