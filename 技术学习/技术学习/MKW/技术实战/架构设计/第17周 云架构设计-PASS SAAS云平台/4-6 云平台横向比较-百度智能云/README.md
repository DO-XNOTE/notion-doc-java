这不是骂人的话是什么？头条、美团、滴滴，以及其他的很多独角兽企业，到底是 BAT 还是TMD？付妍老师要推出另外三个英文字母，叫a、b、c。大家猜一猜，这是哪三家企业？在飞扬老师心目当中，未来会执牛耳者？不好意思，不是企业。它是什么？三个具体的科技的名称，分别是 AI 的a、 big data 的 b 和 cloud 的c。各家互联网公司、各家 IT 平台、各家资讯公司，基本上都达成了一致。未来是什么？是以云为基础架构，以 big data 为平台化，以 AI 为什么？最终的服务为体现，从而实现全自动无人工干预，大量的数据流，高频的数据流的处理，以及强大的基础架构，便宜的基础架构和灵活的调整部署伸缩，这才是未来的方向。基于此，百度云就提出了一个很大的口号，叫百度智能云。它尝试用一个平台，既完成AI，也覆盖对data，同时又提供云平台服务。我们来看一看它长什么样子。这就是百度智能云的总体架构。
