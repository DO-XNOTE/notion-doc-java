那我们来看一下 source 本身而言的话，它就是从外界读取一个流的源头，这个名字你就可以看出来它就是source，这个 source 是后续阿卡这个流里面它要消费的一切流的元素的源头。但是它自身肯定是没有这个数据的，它也是从外界给它灌输的，也就是说它与外界的连接点是通过 source 来产生的。


外界有任何要通过阿卡的流进行处理的数据，它的连接点就在 source 这一点， source 它把外界的各种数据通过自身的一个转化变成一个自己的output，这样它就完成了一个流的输入，也就是流的第一个起点。它有牛的源头，对于牛你不要把它看成一组数据，它是一个无穷尽的数据就对了。


boss 建立起了一个 archive stream 与外界的桥梁，外界数据的桥梁，准确来说，然后从它自身的 plan 来讲的话， source 是第一个环节，然后将它的输出作为下一个处理节点的输入。你在这个地方 source 来建立了一个桥梁，它也是与外界沟通的中介，就是这个桥梁的作用，对吧？然后也是访问 stream 的入口，外界的所有的数据，你要经过 Arkstream 的处理，那必须先通过 source 建立这个桥梁，打开你的流，让后续的处理的这个 pipeline 能够处理它。同时它只有一个输出点，无论你1万件的数据是长什么样子，但是到了 source 以后，经过它的处理作为下一个的入口，这都是统一的标准，可以从很多不同的东西里面去读取，比如说文件数据库，还有一些q，甚至一个固定的 connection 都可以作为它的一个数据来源，通过 source 的转化变成一个流。

它需要有一个分配资源的过程，这就是叫雾化。就是说流的运行之前要先给它分配资源的这个过程就叫雾化，如果没有雾化，这个所谓的 stream 在阿卡中它是无法实现任何工资的。也就是说你只是看得见一个东西，但它并跑不起来，就这么简单，因为没有经过雾化的话，这个流它是不能获得任何系统资源的，它也无法进行运行。


简单来说，阿卡就是要启动流中的 actor 并实际开始执行处理。英语我们有说过卡 x 的处理方式的一种包装，简单来说，你甚至可以把这最肤浅的认为就是一种对 Anchor x 包装的育发糖，让你超出一些更方便，其实你背后都是 x 在运作前面的基础概念里面我已经讲过这一点，但在这个时候物化的过程就表示这些 actor 都分配完毕，要真正开始启动它实现具体的功能，开始运行所有的 terminal 操作。


如果像前面例子中的 wrong with 或者 wrong 这种方法，它就会触发流的雾化，就表示说我这个 graph 已经构建完毕了，现在我要开始真实的运行了，也就是告诉阿卡 assistant 所有的这个牛的 pipeline 或者图已经画好了。


现在我真的要开始运行了，你可以开始给我传输数据，我可以开始做处理了。一个类比可能未必完全准确，但是应该有助于你理解物化这个概念。你可以想象程序员在写代码的时候，通过source、floor、 SYNC 这种东西这三大元素构成了各种各样的pipeline，它这个 pipeline 它只是一个设计图，这个设计图它只停留在纸面上的感觉。也就是说你是知道从概念上我们定义了这样一个pipeline，但是数据它真正的要开始运作，那我们还差最后一个步骤，就是把它真正的实现出来，也就是要把它物化。所谓的物化你就相当于你把工程师画的图纸变成一个实物做出来，在这个地方它就被叫做物化，给虚拟构建出的 pipeline 形成实体，让它整个 pipeline 能够运转起来，这是物化。


所以说你印证这个事情就和 Java 原生的这个 stream 流操作一样，我曾经讲过它必须有三个部分，一部分就是原一步，就是中间的operate，最后一步就是 termination 操作。其实这个层就是一模一样的一个概念，所有的流操作它必须都有最后一个 terminate operation 才能触发真正流的运转，在这个地方无法就承担了这样一个责任，它第一是让这个流开始运作，通过 run 或者 run with，第二就是表示真的要运作，你给我分配好资源，现在我开始运作。

当雾化这个过程在发生的时候，它将为流生成一个雾化值，一个阶段的雾化值可以传递给另外一个阶段。我们把 graph 当中的每一个节点把叫一个阶段，叫 stage source 到floor，它各是一个stage， floor 到 sink 又是一个不同的stage，这个物化成功以后产出的值，它是可以由使用者来决定有什么用途，自己用它来干什么。


每次流在运行时，这个物化的值都会被重新独立的计算，与前面这个值是没有任何关系的，只要这个物化值它每经过一个节点的时候，它就会被计算一次，而且跟前面这个值是没有任何关系的，基本上可以因为是独立的，所以它每次它输出的东西看上去都是物化值，但其实有可能都属于前面那个值，没有任何关系，这就是物化。
