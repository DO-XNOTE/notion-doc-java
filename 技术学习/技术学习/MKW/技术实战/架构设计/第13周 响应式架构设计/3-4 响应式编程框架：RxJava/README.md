在 x Java 里面它有以下的几种，目前来讲一共有五种啊。第一个floyable，这个是最新引入的，它可以发送任意的数据，同时也是支持 reckon stream 和被压的 observable 就是最早已经存在一种类型这样一个publisher，它可以发送任意的数据。它是不支持被压的single， single 它只能发送一个数据或者一个错误，就是它只有单一的用途，不能发送多个，它就只能发送一次，发送完它就没有用 comfortable 不发送任何数据，它很怪，它只是处理两个事件，uncomplete， unerror 这两个事件，别的事件。
