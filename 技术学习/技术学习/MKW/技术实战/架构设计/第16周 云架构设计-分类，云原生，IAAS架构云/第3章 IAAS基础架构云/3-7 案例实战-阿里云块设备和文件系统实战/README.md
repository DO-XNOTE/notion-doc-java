我们看一下每台机器其实后面都挂着一个盘，这个盘默认是系统盘，也就是某一种云盘，可能是普通云盘，也可能是 ESSD 的。什么快速云盘，所谓 ESSD 就是增强型的、扩展型的SSD，大家都知道什么读写都比原来快很多，原因就是因为磁盘是磁介质，有磁头的移动 SSD 是电介质，所以它减少了磁头移动这样一个缓慢的机械操作，所以速度会快很多。那 ESSD 不是一个一个的SSD，而是把一堆 SSD 服务器形成一个分布式系统，然后在分布式系统上面提供出的一个一个硬盘仍然是块设备，仍然是为了结构化数据在做准备。比如说你启动的一些基本盘，或者是数据库的底层盘，或者是什么卡夫卡的底层盘等等等等。

聊完了这种快速的快照，快照有什么用处啊？快照一是做一个当时的这个瞬间的状态记录，之后你所有的修改只会改原来磁盘快照内容是不会被修改的。同时快照通常是什么？是写时复制，也就是啪嚓一个照片，快照就产生了，但当时快照里面是没有任何内容的，但是随着你去不断地更新你原来的磁盘，这些更新的内容它会什么写到原有磁盘，同时它会把老的内容复制到什么？快照盘，那这个时候快照盘里面才真正有了一些内容。同时这些内容什么是当时的那个内容，不会被后来的新内容所篡改，所以快照重点是要一个保护一个时间点的概念，快照可以什么？很方便地进行恢复原来的磁盘，也可以很方便地什么去进行一些什么磁盘迁移等等功能。


有些系统里面分成两个，一个叫快照，一个叫镜像。那快照就是很快的镜像，就是在第一次创建的时候，它就已经完成了数据的复制，所以创建时间相对比较长。比如以 1T 的硬盘为例，它可能需要几十分钟才能完成镜像的创建，但是快照通常是秒级到 1 分钟左右就能完成。
