所谓边界你可以看得很大，比如说国与国，这对应的我们 it 界就是什么系统边界、领域边界，我们做一个完整的系统，比如电商平台 c to seed 一套系统，这就是一个完整的系统，这也是一个完整的领域。在国里面可能会有省、市、县，也就对应着我们什么子系统子域，这可能相当于省的级别，还比较大，那是就是模块诶，已经比较小了。


最后模块下面可能还有聚合实体这种更小的边界，来实现什么数据一致性、业务一致性好，这是什么？第二层边界，那除此以外是不是还有一些边界，大家能想出来吗？这就是什么分层边界？所谓分层边界就是我不能归属在某一个省、一个市里面，它横跨一些省市的。那在实际我们的生活当中，比如说我们的这种某些知名高校，比如像清华、北大，他们并不附属于北京市，他为什么直接由教育部来领导来管理？同样道理，我们有很多个军团，那驻扎在不同的省里面，但他是由什么？是由军委统一进行管理，那一样的道理，我们在分层上面，除了领域层，顶上可能还有些什么？包括应用的边界层、接口层等等，它有点不属于某个细分的子系统，所以那些特殊的层依然也有它的边界。
