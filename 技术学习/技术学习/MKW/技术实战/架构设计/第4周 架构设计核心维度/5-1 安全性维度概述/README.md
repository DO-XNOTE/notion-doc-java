equal fix，这家公司是美国信用评级公司，所有人要在美国买房子，一定要先经过这家评级公司的评级，你才能贷款，才能买房。但这家公司在 2017 年发生一个大的问题，他把一半以上的美国用户，他的个人信息，包含联系人方式、家庭住址、社会卡号、 ID 以及相关的社保的密码等等信息完全公布在网上和论坛里面，导致出现了美国政府和非营利组织的一次大的信用危机。


那 2018 年 Facebook 也犯了个类似的错误，他把他 6800 多万用户的实际信息完全的什么暴露在了公网上？在当时出事的那几天之内，我们可以通过公网很方便地查到所有 Facebook 的用户，他的真实姓名，他的ID，以及一些实际的 Facebook 使用情况，这导致同一年内 Facebook 的股价直接跌掉30%，非常大的一个经济损失。


好，我们再看一看。 2018 年还发生了一个更大的事情，那就是万豪集团信息泄露事情。万豪集团包含像万豪酒店、万怡酒店、喜来登酒店的一个大型集团，它有5亿的用户。在同年，他把所有用户信息以及所有用户的开房信息都被暴露在了公网上，被黑客攻击，导致完全数据泄露。这样的一数据泄露对企业造成了多大的影响我不可而知，但是却造成了巨大的社会动荡，大家可以想象很多国家的高管纷纷落马，原因是什么？原因是在不合适的时间，在不合适的地点，他们出现了开房记录，那同时也有很多的家庭或者婚姻出现了崩溃。原因也很简单，一个丈夫查一查自己老婆的在同一城市，在自己家所在城市的这个开房记录，今年居然开了 10 次房，这里面是不是有被戴绿帽子的嫌疑？就是因为这个原因，有大量的这个家庭和社会的动荡，导致什么万豪集团口碑急剧下降。好，我们再看一看国内有没有类似情况。 2019 年拼多多的，什么拼多多的这个优惠券事件是不是引起很大的轰动？你可以认为它是一个规则引擎的功能问题，当然也可以认为其实是一个安全故障，这次故障直接导致什么？发出了 200 多亿人民币价值的优惠券，在拼多事后通过各种途径去消解和缓解了这次事件导致的金融影响，但仍然损失非常大，口碑损失也非常大。


所以综合来看，安全什么无小事，我们有一句话叫，人在河边走，哪能不湿血，你安全不注意总归会出现问题。那濮阳老师补一下他的下联好不好？就要想脚不湿请买安全套鞋，你必须穿好你的安全套鞋才能够什么防止是吧？出现脚湿的情况对不对？好，那下一节我们就来看一看怎么样从流程上保证什么，我们每个人都穿着自己的安全套鞋。好，谢谢大家。
