那通常当一个架构师要去进行新的站点选型的时候，才会设计它，需要考虑新的数据中心该如何进行选型，如何去搭建一套数据中心，那这个时候你会考虑什么呀？我的数据中心可能需要防止有些黑客来进行数据偷取的。那应该怎么做呢？我是不是要设计一些安全保安技术？那这个安全保安呢？它就有相关的人员安全，怎么保护它的人身安全，同时保证它的权责是可以分离的。
