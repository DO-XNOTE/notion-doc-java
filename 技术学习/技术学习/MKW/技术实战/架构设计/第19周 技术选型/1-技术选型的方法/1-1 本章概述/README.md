首先会探讨什么是技术选型，以及学习技术选型的一个重要性。第二会跳脑，大家在项目里面做技术选型时的误区，这些误区比较的典型，是很多人会犯的。接着来探讨做技术选型的一个步骤，第一步怎么做，第二步怎么做手把手。第四会介绍技术选型的两种方法。第五是怎么样选择一个计数的版本，因为版本也很重要。最后是如果不幸选择了一项不适合项目的技术，该怎么样去补救，以及怎么样平滑的更换项目的技术选型。以上是这一章的一个大致安排，话不多说，我们正课见。
