但是要给大家布置一个作业，分析目前市面上 3-5 个主流的电商平台，并汇总报表分析的维度。比方各个平台的优缺点、运营模式、商业模式、发展潜力、用户体验竞争力、投资者欢迎程度等等。你还可以任意的扩展其他的维度。你想象你现在正在准备买某一家电商平台的股票，


你准备怎么样去投资哪一个电商平台，或者是哪几个电商平台？对于分析的方法，同学们可以借鉴前面介绍的因素加权法，分析你所关心的指标。也可以参考一下网上的文章，看看人家是怎么样分析的。之所以布置作业，主要有几方面的用意。首先，越往高层走，越需要对行业有一个预判的能力，怎么样才能预判，如做大量的分析。二来，很多的面试官他喜欢问一些市场相关的问题，因此了解一下目前电商行业的市场格局总是好的。而且今后哪怕同学们从事其他领域的工作，在面试之前也应该了解一下你所面试的公司在市场所处的位置，公司的定位，公司的愿景等等，这样也可以提升面试的成功率。同时，如果有一个公司，它的定位非常的差劲，公司也没有任何的愿景，这种公司你甚至可以不去面。


第三，相信在座的很多同学未来会走向投资理财的道路，不管是炒股还是买基金或者是其他的投资方式，其实本质都是对投资的目标进行一个分析，并判定趋势，识别风险。经常做这样的分析，也有助于商业思维的发展。话不多说，期待同学们将调研的结果在问答区或者是 QQ 群和我交流。这节课就到这里，谢谢大家。
