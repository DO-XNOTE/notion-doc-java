首先，要在某一专业领域中，对公司内外以及业界的相关资源及水平要比较的了解。第二点，他就说要对部门管理层在某一个领域的判断里产生影响。再往上走，领域就更加重要了。比方 P9 要求是某一个领域的资深专家。

首先，只有真正在某一个领域工作过，才能对这个领域有所了解。有人说隔行如隔山，就是这么一个道理。而有些领域甚至需要数年甚至更久的时间才能有所沉淀，但是一旦积累完成，就到了收获的季节。只要这一个领域的前景发展良好，到哪儿都会很吃香的。比方和我合作的一个室友，他一直从事供应链相关的工作已经快 20 年了，在这方面的沉淀是非常深厚的。在阿里的职级是P9。有人可能要质疑了， P9 还搞什么合租？这里解释一下，他之前一直在厦门工作，为了事业抛家弃子，只身一人来到了杭州。这一点和大木一样。而杭州它是有限购的，不是一来就能买房的，所以就合租了。当然了，扯远了。


这里我们再说一句题外话。建议同学们在学习以及跳槽的时候，不光要看技术方向，也得找好业务领域。并且一旦找好合适的业务领域之后，就应该持续的在这个领域深耕。有些同学跳槽就是东一锤子西一棒。比方现在华为干运营商相关的业务，再跳到阿里干电商，再之后又跑去腾讯干社交。虽然这些企业都是名企，但是这种跳槽方式并不是利益最大化的一个选择。因为你前面的工作经验，相关的领域沉淀都没了，一旦跳到新的公司，又得重新积累。从华为跳到阿里，你得重新积累。从阿里跳到腾讯，你直接没有干过。社交又得重新积累。我们回到正题，来看看课程的安排。
