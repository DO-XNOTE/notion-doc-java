首先要学习编程语言， Devops 有一些开发技能，对不对？要了解不同的操作系统概念，像socket、posix io 等等都需要有一定的认识。要了解如何管理服务器，比如要了解各种各样的操作系统，了解如何使用终端，比如 8 小 v 母 Nano 的。了解网络安全以及各种协议。


了解如何搭建各种服务器，比如反向代理服务器、缓存服务、存服器、负载均衡服务器、 web 服务器等等。要学习基础设施及代码，比方容器配置管理、容器编排、基础设施供应等等。也是目前市面上的热门，在咱们课程也有所覆盖。了解一些 CICD  的工具，了解如何监控软件以及基础设施，包括了基础设施监控以及应用监控，日志管理。有关监控在本套课程的监控章节会详细探讨。


最后还要了解云端设计的一些模式，比方一些云服务提供厂商。好。讲到这里，相信不用我说了，如果每一个都去学习一下，可能学几年都学不完。在这里作者做了一个标记，紫色是作者个人的选择，绿色是一些候选项，和紫色达到的效果是等价的，灰色是非必选的功能，你可以在用到的时候再说。最后，没有对号则是作者不建议的技能。这样一来，要设计的内容会少很多。同学们可以对照图看看目前已经点亮了哪些技能，哪些是目前还没有掌握的。
