首先到这个地址下载Jenkins。 Jenkins 提供了两种版本，一是稳定版，这种版本可以获得相对长期的技术支持。二是定期发布版本，每周发布一次，这种版本可以获得更新的功能。实际项目中，如果没有特殊需求，建议使用稳定版。我们课上使用当前最新的稳定版本，二点二七、七点二进行讲解。


Jenkins 提供了多种安装包，你可以根据自己的操作系统进行下载，也可以使用这里的通用包。课上我们使用通用包进行安装，点击就可以下载通用包了。如果你的网络不好，可以使用这个地址进行下载。下载完成之后，只要在安装包所在的目录右击打开终端，执行 Java 杠jar，跟上 Jackie 袜包的名称，就可以安装 jackiness 了。当然了，你也可以把外包调到 Tomble cat 之类的 web 容器里面去执行。比方调到 tomcat 的 webapps 目录像，然后到 bib 目录启动 Tomble cat 也是可以的。
