我们希望用 Jenkins 实现这几个目标

单元测试

项目构建、 

Docker 镜像的构建，

并把应用部署到Kubernetes。

要想实现这 4 个目标该怎么做？方法非常多。先来看到第一种方式自由风格的项目。点击左侧的logo，来到 Jenkins 的首页，点击新建item，就可以创建 Jenkins 任务。填写名称，比方说 Jenkins Eureka server1。名称是任务的唯一标识，项目类型选择 freestyle project，表示自由风格的项目确定。这样就会弹出一个任务的配置界面。 general 是任务的通用配置。源码管理选择Git，填入项目的 Git 地址，
