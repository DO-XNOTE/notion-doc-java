大家好，我是大木。俗话说得好，工欲善其事，必先利其器。一款良好的项目管理工具是我们管理项目的一大助理。这节课我们来对市面上流行的项目管理工具做一个对比，从而帮助同学们在企业中选择。


所谓的项目管理，主要有两个方面，一是需求管理，二是 BUG 跟踪。


这两点是每个项目都要去考虑的。好的项目管理工具可以帮助我们更加高效的跟踪项目的状态。目前市面上有非常多的项目管理软件，他们做了一个整理，并分成了三类。

针对自见型的项目管理软件，做了一个比较详细的对比来看一下。从中可以看到， JIRA 是商业授权产品， read mine 是开源产品，而禅道既有开源版，也有商业版的功能更加强大一点。从功能来看，这几款产品的差别并不大。而至于第二类以及第三类的项目管理软件，由于有一些产品，它不提供 demo 测试，所以不太好做对比。如果同学们决定采取这种方案，需要自己调研一下。


课上，大木将会结合 redmine 进行讲解。当然了，实际项目里面，你用其他的软件也是 OK 的，比方用 teambition 或者是用 JIRA 都行。在做选择的时候，可以结合使用体验、业界流行度、成功案例等等因素，并亲自上手玩一玩，试一试，最终敲定一款工具，在项目中落地就 OK 了。可以这样，不管哪一款工具，功能都是非常强大的，最关键的是要能够让工具和你的项目契合起来。OK，这节课就到这里，谢谢大家。
