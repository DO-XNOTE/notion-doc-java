大家好，我是大木。前面几节课我们介绍了 Git flow 以及 GitHub flow 这两种代码分支策略。除此以外，还有很多其他的代码分支策略，比方 getlab flow、 one flow，阿里的 Aone flow 等等。但是由于这些代码分的策略在国内其实并不是特别的流行，所以课上我们就不去展开探讨了。
在这里我找到几篇不错的文章。这篇文章介绍了各种代码分支策略的特性，有创个 based git flow， Aone flow， Exe flow 的，写的还是挺细致的。这篇文章是 Git flow 以及 GitLab flow 的对比。这是一个英文文档，里面详细对比了 Git flow 以及 GitLab flow。此外还有这篇文章，分别介绍了 Git flow、 Github flow 以及 GitLab flow，感兴趣的同学可以看一看。这是在上，这是在项目里面选择代码分支策略的参考。这节课就到这里，谢谢大家。
