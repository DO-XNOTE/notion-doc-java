这三者之间有什么区别？先来说一说hosted。它表示这是一个宿主仓库，这种仓库就是我们平时所谓的仓库，它用来存放企业内部的各种依赖。比方你有一个价包，想要和团队的其他成员分享，或者和其他的团队分享，都可以发布上传到 hosted 仓库里面。

第二是 proxy 仓库代理仓库。这种仓库可以设置一个远程仓库的链接。当用户向 proxy 类型的仓库请求一个依赖的时候， proxy 仓库会先在本地查找，如果找不到，就会从远程仓库下载并返回，下一次如果再请求相同依赖，就会直接从本地返回了。也就是 proxy 仓库是用来代理其他远程仓库的。


第三是 group 仓库聚合仓库，用来将多个 hosted 仓库或者是 proxy 仓库聚合到一个 group 里面。这样在使用的时候，只需要配置 group 仓库的地址，就相当于配置了 group 下面的所有仓库了，从而简化我们的配置。这是几种仓库类型？聊完仓库类型之后，
