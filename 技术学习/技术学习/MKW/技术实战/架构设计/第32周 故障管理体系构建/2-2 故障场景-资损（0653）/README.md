第一种就是实际的资损。实际资损其实都有哪些？比如因为故障导致我们少收了客户的钱，这就意味着客户占了便宜，实际上公司也出了损失。少失了客户的钱，就意味着我们本身的成本就把被损失掉了。


第二就是因为故障导致多给了客户的钱，客户提供给我们某些东西，给客户相关的钱，这时候我们多给了他钱，就意味着拿到了一份东西，但是就付了两份的钱，这也意味着我们本身产生了损失了。第三个就是客户用我们的服务，比如 c 端用户去做生意，或者是本身要收一些钱，平台，尤其是像平台这种，客户来了，用平台向 c 端用户去收钱。而因为故障导致客户亏损的钱，不是其本身经营出现问题而亏损的钱，而是因为我们的故障而他亏损的钱，其实我们是要赔给客户的，这一类就是我们的资损。


第四类就是因为故障，公司直接亏了钱，这一类也是我们的实际自损，这四类都属于实际自损。还有第二就是潜在自损，什么是潜在自损？实际上钱好像是没有赔，但是实际上我们应该比这挣得多，这就是因故障导致公司应该挣没挣到的钱，这就是一种潜在自损。第二种就是因故障导致客户应该挣没挣到的钱，从实际角度去看，的确是资损没有产生，但是从潜在资损去看，其实是产生。所以这两类就是我们常规的一个资损的概念。总体上去看，任何的资金损失或潜在的损失都成为资损。在本身的故障中，自损是怎么产生的？
