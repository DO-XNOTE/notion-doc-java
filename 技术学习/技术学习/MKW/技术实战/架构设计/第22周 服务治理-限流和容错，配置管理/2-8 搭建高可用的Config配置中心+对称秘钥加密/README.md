hello，目前的各位同学们，大家好，我是姚半仙，咱这一节呢，开始跟同学们去了解一下我们 spring Claude 他自己的亲儿子原创的 config service 组件。那我们这一节的目的是搭建一个高可用的 config Server 组件，但是这个目标咱只能完成一半，为什么呀？因为所谓高可用，实际上就是说使用我们的注册中心，把咱的 config  Server 自己作为一个服务注册上去。
