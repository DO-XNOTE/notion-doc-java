# 2-9 沟通技能-有效反馈

好的各位同学，我们已经学习了合理表达和深度倾听的章节，通过合理表达的章节，我们知道了如何去避免语义规则造成的一些表达的障碍，让对方更好的去理解我们。同时我们也可以通过结构化的表达思维，让自己的表达逻辑更加的清晰，更加的清楚。同时我们也学习了深度倾听的三二原则和倾听地图，能够让我们更好的了解对方的情绪，了解对方的需求，并且很快的找到对方表达信息当中的事实。

就是有效反馈的技能。有效反馈有些时候是出现在沟通的过程当中，有些时候也会出现在沟通达成共识以后，共同去执行落地共识的这样的一个过程当中。很多时候我们在跨部门做一些项目合作的时候，常常就需要用到有效反馈的技巧来让对方知道我们反馈的内容到底意味着什么，也让对方能够更好地接受我们的反馈信息。那么我们有的时候在反馈的过程当中可能会造成一些冲突和矛盾，这很可能跟我们反馈的方式以及反馈的技巧是有关的。这个章节我也会去讲我们如何用更好的建设性反馈的氛围来进行反馈的冲突的。

甚至我们有的时候会对我们的客户进行相关的反馈，那么我们一般会把有效的反馈分为两种，一种的话叫做积极性的反馈，一种的话叫做建设性的反馈。简单来讲，积极性的反馈就是表扬、赞赏和认可，而建设性的反馈其实就是给别人提意见，指出别人行为当中的一些不足或可以完善的地方。那么如果我们说的再通俗一点的话，建设性的反馈其实就是一种批评，就是一种指导，让别人去改变、完善他不好的行为。


我们先来聊一聊，积极性的反馈，就是表扬，就是赞赏，就是夸别人。

有些同学听到这个可能会想表扬别人，夸别人我还不会吗？我经常表扬别人，我在工作当中会经常夸我的合作伙伴，夸我的同事。那么好，问题来了，请问同学们，我们在职场当中对别人进行积极性反馈的时候，我们的目的是什么？在积极性反馈的过程当中，你想达到什么效果？我认为我们在职场当中对别人进行积极性反馈，一定不是单纯的因为我们心情好，对吗？也不是想去讨好别人，去奉承别人，对吧？可能我们在给别人进行反馈的时候，有一些反馈的内容是想拉近和对方的距离，保持双方的关系。但是很多时候这些并不是我们给别人做出积极性反馈的所有的目标。很多时候我们给别人做出积极性的反馈，更多的目标是想强化他某一种比较好的行为。比如说我们跟某个同学在配合项目的时候，他工作很积极，态度很认真，而且每一次在跟我们解决问题的时候，都能够第一个冲到最前线。


那我们对他进行积极性的反馈，除了是想表扬他，除了是想跟他保持一个良好的关系，建立一个未来持续合作的这样的一个状态之外，更更多的我们是想通过积极性的反馈来怎么样来去让他认识到自己一个好的行为，甚至能够保持住这样的一个合作的态度和状态，这个才是我们在职场当中做积极性反馈的一个非常重要的因素。


那么我们在职场当中的话，会发现很多时候我们做了一些积极性的反馈，其实并没有起到这个作用，不仅没有起到这个作用，很多时候我们连让对方接受我们的积极性反馈，来认为我们的积极性反馈是真诚的，是真实的，都做不到。为什么呢？因为很多时候我们做积极性反馈的时候是这么做的。最近可以啊，干得不错。


好了，各位同学，想一想你们在工作当中或者是在生活当中，有没有对你们身边的人做过类似的赞赏？这样的一种积极性的反馈，你认为对方会怎么想？假如我收到了别人的类似于这样的一个积极性的反馈，我的第一想法是什么呢？我觉得这是对方的一种寒暄，一种没有经过深思熟虑的一种客气。那么这样的一种寒暄，这样的一种客气，我根本就不会放在心上。而且有的时候别人突然跟我说元宝最近干得不错，表现的挺好，我反而还会感觉很疑惑，为什么呢？我到底哪里干得不错？我到底哪里可以？我最近的工作状态跟以前是一样的，我也没有做出什么特别突出的事情，为什么突然来跟我说这样一句话，你是不是别有用心，对吧？所以这样的一些积极性的反馈，我们来讲的话，它其实是没有太大的意义的。


那么不管是对于我们来进行积极性反馈的人，或者是接受积极性反馈的人来讲，这都是没有太大意义的。那么我们如何做才能让我们的积极性反馈变得更有意义，也能让对方更加的接收我们？来看第二句话，每次我发布上线需求，你都是第一个响应，并且主动和我确认线上环境，你真挺靠谱的。


好了，如果别人对我们用这样的一种积极性的反馈，我们的感觉是什么呢？首先，我说一说我的感觉。我的感觉，第一，对方能够描述我在工作当中的一些具体的行为和事实，我认为他对我是关注的，起码他是知道和了解我的工作的。第二呢，他对我做出了这个积极性的反馈，让我明确的知道了他为什么要夸赞我。他并不是跟我客气，也不是跟我寒暄，他真的是就工作方面的实际情况来跟我进行沟通的。第三点，对我做这样的一个积极性的反馈，让我明确的知道了我自己到底是哪些行为，哪些事情做得比较好，让他是认可的。那么在未来的工作当中，我如果想提高我自己，保持我自己，我也知道了我自己应该在哪些行为上，哪些方面上去进行保持。所以这样的一个积极性反馈的话，是走心的，是真诚的，他不是一个客气，不是一个寒暄，也不是有其他的别有用心的想法，对吧？如果我们在表扬别人，夸赞别人的时候，能够把我们表扬他的具体的客观的事实和行为描述清楚，那么我们的积极性反馈就已经完成了第一个步骤了。


我们来看一下这个人叫做赫勃克勒赫，他是美国西南航空公司的CEO，曾经他以自己高超的管理技能和非常突出的沟通技巧，帮助西南航空公司成为美国最赚钱的航空公司，他也深受他的员工的爱戴和尊重。

在他过生日的那一天，他的员工为他把当地的报纸的头版给包了下来，对他进行了一些赞赏和积极性的反馈。那么我从中去截取了一段，大家可以看一下，能够记住每位员工的名字，在感恩节帮助搬运行李，能够静静地倾听员工说话，允许大家穿自己的短裤和休闲装上班，和我们一样骑着摩托车上班，OK。这些其实都是他的员工对这位 CEO 的具体的行为和事实做的描述，对吧？他们并没有简单的向这位 CEO 说你是一个伟大的CEO，你是一个特别棒的CEO，对吧？那样的一个积极性的反馈方式，其实并不能让别人真正的赞同，或者是知道大家为什么尊重他，为什么爱戴他？而这些在工作当中一件件的小事，一件件的一个具体的行为，真的能让大家了解他，知道他到底是一个什么样的人。


而对于这位 CEO 来讲的话，他也通过这样的一个积极性的反馈，这样具体的反馈内容，真正的了解自己到底是哪些方面做得好，到底自己是因为什么让他的员工那么的爱戴他，那么的喜欢他，那么他在未来的工作当中也一定会继续努力的去保持这些行为。


大家同意那这些的话只是描述了他的一些具体的工作当中的行为和事实，他的员工在这些事实行为背后又加了一段内容，让他更加的认可这些员工对他的赞赏，他接受到的有效的积极性的反馈也更加的详细，这些员工加的是什么呢？他们家的是红色的字体，这种感觉我们好像是一个大家庭，鼓舞了弟秦弟兄们的士气，让大家觉得每个人说的都很受重视，这让我们更加投入到工作中去，我觉得我们和 CEO 一样都是普通人，那这些红色的字体到底代表了什么呢？它代表的是这些蓝色字体所带来的什么积极的好的影响，对吗？因为它能够记住每位员工的名字，所以这种行为让我们感觉大家是一个大家庭，对不对？所以这些蓝色的字体引出了红色的影响，这样的话这位 CEO 就更能体会到他的员工对他的赞赏，对他的积极性的反馈，到底是因为什么？是因为他的具体的行为带来了一些比较好的影响，那么他就更容易接受，并且他更能清楚明白自己的哪些行为带来的哪些好的影响是受欢迎的，这对于积极性反馈的有效性以及可接受度是非常重要的。


通过这个案例，我们就能引出我们今天的积极性反馈的法则，叫做 BRA 法则。那么 b 的话代表是具体的行为，代表的是事实。

我们所有对别人这样的一个夸赞，这样的一个积极性的反馈，都要基于事实，都要基于他的行为。脱离了事实，脱离了行为，只能让我们的积极性反馈听起来是空中楼阁，听起来是客气，是寒暄，那这个事实的行为背后带来的影响是能够加深这个反馈的深度和有效性的，那么有了行为和影响，最后我们再加上感谢，再加上感激，那么会让对方觉得我们真的是真诚的在夸赞他，他也能够真的认可和知道自己的哪些行为、哪些影响真的帮助到了别人。所以这样的一个积极性的反馈，我们认为才是一个完整的、有效的积极性的反馈。

这是具体的事实和行为。有了具体的事实和行为之后，我们看第二句话，由于你的认真和负责，我们研发团队每次都能很透彻的理解需求，进而高效的进入开发流程，顺利的完成研发工作，这是什么呀？这是刚才的那个具体的事实和行为带来的有利的影响，对吧？这个叫做i。最后大家都认为你是一位认真负责、有责任心的同学，很感谢你为研发团队带来的帮助，这是什么了？这是基于事实的行为和积极的影响背后我们带来的感谢和感记。这个例子的话，就是一个 BRA 的比较完整的，比较完善的一个具体的反馈的这样的一个法则。大家可以通过这个例子再去详细的了解，详细的去研究一下 BIA 反馈它到底应该怎么做。实际上 BIA 的反馈不仅能用在工作当中，也可以用在生活当中，比如说我们和我们的父母进行沟通，我们想对父母做一些积极性的反馈，表示感谢，表示感恩。但是由于我们的反馈方式都是比较简单的，都是不那么具体的，所以我们的效果往往不是很好。父母对我们的一些感谢和感恩也只是进行简单的回复。就比如说我跟我的父亲，很多时候他为我做了一些事情，我就会跟他说谢谢，谢谢你对我的关心，谢谢你对我的爱护，那我的父亲的话也是会简单的回复几个字，那么我们的积极性反馈和回应的话就是这样的一个状态。那当我学了 BIA 的反馈之后，我想尝试一下这个方式来对我的父亲进行积极性的反馈，所以有一次的话，我就会对他的一个行为做了一个详细具体的描述，并且告诉他他所做的这些事情能让我更安心的工作，对我在工作当中去学习，去成长有很大的帮助。那么最后我对他进行了非常真诚的感谢和感恩，没想到我用这种方式给他进行了积极性反馈以后，他也通过短信的方式给我回复了很长的一段话。通过这个 Bia 的反馈的话，我跟他之间感觉到好像关系又更近了一步，我们互相彼此对方的一些行为和行为带来的影响也更加的认可了。所以积极性的反馈看起来很简单，但是它的这种实用性和有效性还是真的真的是值得我们去尝试的。我也希望同学们的话，在今天学完积极性的反馈以后，你能够找到你的某一位同事，你的某一位朋友，甚至是你的某一位家人，尝试去使用这个反馈的模型，对他们进行一次反馈，也许你会在今天得到一个意想不到的收获。


好了，我们讲完积极性的反馈，接下来就要去讲一讲建设性的反馈了。


如果说积极性的反馈是对别人进行夸赞和表扬，那有的时候，即使我们夸赞的表扬的不是那么好，说了一些话也不是那么有逻辑性，也没什么太大的问题。因为我们中国有一句俗话叫做伸手不打笑脸人，对吧？毕竟我们是夸别人的，但是建设性的反馈可就不一样了。

建设性的反馈是什么呢？建设性的反馈是给别人提建议，是指出别人工作行为当中的一些不足，甚至是批评别人，所以我们一旦建设性反馈做的不够好，或者是我们的建设性反馈的表达方式不好，很有可能会引起对方的什么呀抵触心理，甚至会引发一些冲突和矛盾。所以怎么样去避免这些冲突和矛盾的话，就显得非常的重要了。


我们举一个例子，小牧他有一个非常好的朋友，有一次小牧做了一个产品报告，他的这个朋友看完了小牧的产品报告，就怀着一颗非常想帮助小牧的心情来跟小牧做了一次建设性的反馈。他的这个朋友是怎么说的？小木，你这个设计思维太老套了，你应该删掉老套的设计，反正缺乏创新，到时候你得负责的跟我比，你的这个设计有点水，这个设计上新以后肯定得失败。


这些都是建设性反馈的句子，它其实它的出发点是好的，是想帮助小木的，但是小木的心情是什么呢？如果你是小木，你怎么想？我想大多数的同学如果听到了这样的一个建设性的反馈，心情可能跟小木也是一样的，像一拳打死这个同学对不对？你凭什么用这种方式跟我说话？你说这些话到底是什么意思？所以很多时候，即使我们在给别人做建设性反馈的时候，我们是抱着一个帮助他，抱着一个非常善良的心态去进行的，但是由于我们表达的不够好，很可能也会引起冲突和矛盾。


那为什么呢？因为我们每个人都是有自我认知的，我们对于自己的话会有一个自己的这样的一个判断，我们到底是一个什么样的人，我们在什么方向上是有优势的，同时我们还会去做印象管理。

简单来讲就是我们会给自己一个人设，并且我们会希望他人都能够根据我们的人设来判断，我们来看我们。


那如果别人就我们进行了一些评判或者是一些反馈，和我们自己的自我认知或者是印象管理是冲突的，那这个时候我们就会出现沟通的防卫。而且与激烈的争论相比，直击要害的批评和揭露引起的防卫心是最严重的。意思就是有的时候我们去大声的去做一些争论，去吵架，反而能够激起我们去抒发自己的观点和情绪。但是如果别人直击要害去说了一些我们的一些不足，而且这些不足和我们的自我认知和印象管理又是有非常大的冲突的时候，这时候爆发的矛盾可能是更加的严重的。


右边的话，我举了 5 个冲突式反馈的一些方式，这些冲突式的反馈方式的话，不仅不能够达到反馈的效果，反而会引起对方的敌视，会加剧双方关系的恶化。比如第一个负向评价，第二个支配导向，第三个冷漠，第四个俯视，第五个五段。那为了去解决冲突式反馈的这五个行为，我们也有对应的一些解决方案，那就是我们用建设性反馈的方式去进行反馈，那包括什么呢？描述事实、问题导向同理平等协商这样的一些反馈的方式的话，会让对方更加容易接受，而不是鄙视。

第一个，负向评价，你这个设计思维太老套了，这就是一个标准的负向的评价，他会让对方觉得你是站在我的对立面，你是有一种高高在上的感觉，对我进行了一个负向的评价，而且这种负向的评价并没有说出具体的事实和依据，这个其实和我们在第二章合理表达当中的标签化评价是有点类似的。


那么为了我们去避免这样的一种冲突式的反馈，我们可以用描述事实的这样的一种反馈方式，我们可以说这套方案很多细节和前几年是一样的，那么我们说了事实，我们并没有做一些负向的标签化的评价，对方听到这句话，他可以自行的去进行查看对比，然后得到一些结论来去改变他对对于这个设计思维的看法。


那第二个你应该删掉老套的设计，这是一种支配导向，是直接对方进行命令式或者是指导式的这样的一种沟通方式，那这样的方式的话很有可能会引起对方的抵触，因为他会认为你凭什么教我做事情，对不对？为了避免这样的一个支配导向带来的敌视的话，我们会用问题导向的方式来代替，比如更加创新的设计方案对你有什么好处？你觉得如果要执意的使用你现在的设计方式，可能会带来什么样不好的后果？如果我们在你的这个设计方式之上加一些创新，会有什么好处？这样的一些问题的话，不仅不会引起双方的冲突，不会引起他的敌视，反而有可能会引发他深入的思考，也有可能的话会我们迂回式的来让它改变它的这样的一个设计方案，这是建设性反馈的一个方式，叫做问题导向。


第三个，反正缺乏创新，到时候你负责这是一个什么呢？让对方觉得很冷漠，你是站在我的对立面对我做了一些内容的评判和描述，而让我感觉到你对我的评判和描述的话，好像是在看笑话，并没有要帮助我的意思。所以对方内心的深处一定也是怎样觉得不接受的或者是鄙视的，所以为了避免这样的一种敌视的话，我们更加倾向于用同理心的方式去进行表达。比如我以前也因为缺乏创新惨遭失败，后面怎么样我们再去进行其他的描述，那要比我们直接告诉他缺乏创新，到时候你负责更让他容易接受，因为我们跟他建立了一个比较良好的同等关系的这样的一个沟通的氛围。


第四个跟我比，你这个设计有点水，这是一种高高在上的俯视的这样的一种反馈方式，让对方觉得我们比他强，所以我们在指责他，我们在嘲笑他，让他觉得很不舒服，引发一些沟通当中的敌视。为了解决这种敌视的话，我们更加倾向于采取平等的沟通反馈方式。我有一些经验，你愿意的话我跟你分享一下，这就会让对方觉得双方的沟通的关系更加的平等，而且这是一种帮助，让他更容易接受。


第五个这个设计上线后肯定失败，这是一种非常武断的，比较粗暴的这样的一种负向判断，这种武断的负向判断的话，不一定能让对方接受他，反而还会有所反驳。因为很多时候人们都是更加相信于自己的判断，相信自己做出来的东西，所以会引发敌视。那我们更加倾向于用协商的方式去解决这个问题，比如现在上线可能不妥，一起来完善一下好吗？这种方式的话会让双方的沟通氛围更加的缓和，在提出建设性反馈的时候，让对方能够有更加多的思考的这样的一个空间，这样的话也能让他更加的接受。其实建设性反馈有五种方式，这五种方式的话要根据我们不同的反馈的场景和反馈的目标，我们去灵活的选择和使用。就像我们在第一章里边说的，沟通是没有办法复制的，同样建设性的反馈也是没有办法复制的。很多时候我们需要根据我们建设性反馈的内容和建设性反馈的对象以及当时的场景来选择最合适的。这个是需要同学们在你们的日常沟通的建设性反馈当中多加练习，多加注意，有意识的去提高这方面的能力和方法的。

你最近提交的需求文档出现了 3 次数据错误，其中有两个文档缺失了数据报表，这是具体的行为和事实。OK，我们说了具体的行为和事实之后，我们要说行为和事实带来的不利的影响。需求文档有错误会导致我们开发过程效率变低，甚至重复返工，对你们提出的产品需求的完成也有很大的影响。这个地方要注意一下，这个负向的不好的影响，不仅仅是对我们开发过程造成的影响，也对他们也就是小木他们团队提出的产品需求完成也有很大的影响。所以我们可以看到在建设性反馈当中我们的影响，负向的影响不能仅仅站在自己的角度去描述，最好这个影响对他本人，对他所在的团队的话也有影响，他才能更加足够的有重视程度去看待这个问题。


最后我们描述了行为和影响之后，就要去对他做一些期待了。我以前也因为工作不严谨出现过失误，后来每次发布文档前都会复查几遍，有时候也让其他人帮忙看看，这样更容易检查出错误。我觉得你也可以尝试一下，会有效果的。这一段话的第一句，我以前也因为工作不严谨出现过错误，这是什么呀？这是一个同理的状态，我们又要站在对方的角度跟他拉近距离，建立一个双方平等的沟通的氛围，反馈的氛围。然后我们来描述我们自己是如何解决这个问题的，而不是直接支配型的或者是指令型的去让他怎么做，那这样的话他可能会更加容易去接受，最后我们会去告诉他，这样更容易检查出问题，我觉得你也可以尝试一下，我们只是怎么样跟你是平等的，让你自愿的去选择去尝试。那这样的话对方可能会觉得更加容易接受，会认为我们确实是站在他的角度为他去好了。


这个案例就是一个比较标准的建设性反馈了，希望大家在学了建设性反馈以后，你们未来再去跟你们的同事，跟你们的合作伙伴去做一些反馈的时候，能够讲得具体，讲得明白，能够跟他在反馈的过程当中更好的有同理心，更好的拉近你们的沟通氛围，让他更容易接受，更容易认知到自己到底是哪里出了问题。
