# 2-6 沟通技能-深度倾听（1833）

几乎所有的同学都不约而同地提到了我的沟通能力不行，主要是因为我的表达能力不行。我的表达能力不行的话，主要是因为我没有办法把我的思想表达得很清楚，或者我再去表达的时候对方听不懂等等，都是这样的一些问题，但是却没有任何一个人跟我提到过倾听的问题，也没有任何人跟我说在沟通的过程当中，自己的倾听能力是有问题的，那说明大家对于自己倾听的这种能力的认知还是存在一定的误区的。


其实倾听在整个沟通的过程当中可以说是最重要的，它的重要性的话有的时候甚至比表达还要重要，因为我们想通过沟通的方式去了解别人，去改变别人，甚至的话是改变别人的行为。但是我们想一想，仅仅是通过表达的这种方式去让别人改变很难，因为我们可能只顾着自己去表达，只顾着站在自己的角度去告诉别人我们认为有用的信息，我们认为对的道理，但是我们却没有通过倾听的方式忽略了通过倾听去了解别人，去真正的感受别人，如果我们都不了解别人，也没有办法去感受别人，我们又怎么能够得到一个好的沟通的结果呢？我们又怎么能够通过沟通的方式去影响别人呢？所以倾听很多时候比表达更加的重要，而且正是因为我们绝大部分的同学都认识不到倾听的重要性，所以我们在倾听这一项上的能力才会更加的薄弱，才会更加的亟待的去提升。


不认识到倾听的问题，这本来就是一个很大的问题。在这个章节的话，我们会给大家讲倾听的三尔原则，包括倾听的地图，也希望通过三尔原则和倾听地图帮助大家能够准确的去认识倾听到底是什么，倾听跟我们过去理解听到底有什么不一样，另外我们也希望通过倾听地图帮助大家了解在倾听的过程当中，我们到底要做什么，怎么样去区分我们得到的信息，通过这些信息的区分和拆解，更加的去了解别人。

其实我觉得它可以反映我们现代人在沟通的过程当中倾听的一种状态。那我们来看一下这个字是怎么组成的？它的左边是一个口，右边是一个金，那我认为这个字的话它代表的意思就是大吃一惊，用口，用嘴巴大吃一惊什么意思？就是我们很多人在倾听的过程当中，总会不断的用嘴巴去打断别人，不断的用嘴巴去插话，好像不表达自己的观点，不插话，不说自己的相关的信息，就不能展现自己的能力一样，所以根本就没有办法去听别人在说什么，也根本就没有办法通过倾听去了解别人真正要表达的这个信息背后的意思到底是什么。这就是我们现代人在沟通的时候倾听的一个常态。


那么我们再来看一下真正的听它到底是什么意思？真正的听它背后到底是什么意义？我们就要去学习一下我们的古人了，古人在造字的时候真的是非常非常的聪明，我们来看一下古代的这个繁体字的听，把这个字拆解一下，右上角是一个10，下面是一个4，这个 4 把它竖起来就是什么呀？就是一个木，代表眼睛下面一和心，整个它的右边的话就是十木一心，十双眼睛一个心，对吧？这说明我们在听的过程当中要用什么呀？要用眼睛去观察，要用心灵去感受，它的左半部分是一个耳朵，下面加一个王，表明在听的过程当中谁是王？耳朵才是王，耳朵才是最主要的。


所以这个繁体字的听它里边有没有口，有没有嘴巴，完全是没有的，没有嘴巴没有口，所以听这件事情跟嘴巴完全没有任何关系。美国有一个非常著名的女性的CEO，叫做弗朗西斯塞赫尔本，有些记者的话在采访她的过程当中就问了一个问题，说你在管理的工作当中做得这么的优秀，这么的出色，那你到底有什么技巧？有什么秘密？弗朗西斯塞赫尔本在回答这个问题的时候说，我其实没有什么太大的技巧和秘密，我只是说在跟别人沟通的时候，我做到了一件事情只有两个字，这两个字就是克制。


因为很多时候我们当我们在沟通，去倾听别人的时候，当别人说了一些我们不同意的观点的时候，我们会做什么呢？我们会立马说话，立马反驳，那当别人说了一些让我们质疑的内容的时候，我们会做什么呢？我们也是会立马说话提出问题。当别人说了一些惹怒我们的问题的时候，我们会做什么呢？同样会说话，会反驳别人。


而以上的种种的话都是阻碍我们跟别人建立联系，阻碍我们了解别人，阻碍沟通氛围的行为，所以我真正的技巧和秘密就是在跟别人沟通的时候，克制自己，不要随意的去打断别人，不要随意的去说话，真正的做到倾听。所以通过美国的这个女性的 CEO 弗朗西斯塞赫尔本，我们可能也能或多或少的去理解一下，倾听对于沟通来讲它的意义是什么？倾听它到底是什么？深度倾听它的本质到底是什么呢？

对方可能会表达一些公开的观点，比如一些信息，一些描述，对吧？对于一些事实的澄清等等，这是对方表达的公开观点，但是对方除了用语言去表达一些公开观点之外，它还会传递非常多的非语言的信息，我们在第一章的内容已经学到了非语言的信息包括什么来着，包括神态、表情、动作、语音、语调，对吧？除了这些非语言的信息之外，可能在整个沟通的场景之下的一些物品，都有可能是沟通过程当中要传递的信息，对吧？这些信息其实是对方向我们表达的什么呢？这些信息可能不仅仅是表达他说出来的内容，可能更多的是这些内容背后隐藏的他的情绪、他的情感，而这些情绪和情感很可能是受到了他的需求的影响。


通过以上的这样的一个分析，我们最后把深度倾听的这个意义叫做站在对方的立场上，不只听取公开观点，更需要感知语言背后的情绪、情感和需求，这个才是我们对于倾听的本质的这样的一个描述，重点的词汇是什么呢？也就是说我们在沟通的过程当中，去倾听别人的时候，不是仅仅听了他在说什么，他表达了什么内容和信息，而更多的话是要用眼睛观察他的动作、神态、语言等等相关的非语言信息，甚至要用心灵去感受他的情绪和情感。最后我们还要多用耳朵去听他表达意思的背后到底还有什么样的一些深入的需求。

我们只有了解了倾听的过程，才能真正的知道在倾听的时候需要注意什么，需要在哪些阶段，哪些方面去提升自己的能力？很多人的话都会认为听和倾听是一样的，它是同样的一个意思，其实听和倾听的意思的话会有非常大的不同。听的话是指声音能够通过一些媒介传递到我们的大脑当中，我们只要能听到声音，这都叫听，它是人的一个生理层面的反应，也就是说大部分的声音只要传到我们的大脑里，我们能够听到，都是可以叫做听到了。比如现在我们的窗外有一只小鸟飞过去了，声音传到我们的耳朵里了，我们就叫做听了。那现在我们的外面呢？有一群人在说话，他们的说话声传过来，我们听到了，这个就叫做听。


而倾听就更为复杂了，倾听的含义就更深了，就是我们刚才讲的倾听是真正的去理解表达的这个人，他背后的一些情绪、情感和需求，它不是一个简单的这个声音传递到我们的大脑，更多的是声音传递到我们的大脑之后，我们怎么样去分析，怎么样去理解，并且结合我们观察到的和我们感受到的其他信息进行一个综合的判断。


这个才叫倾听的话，它有哪些过程？首先第一步的话要保证我们能够听到，这个听到的话有两层的含义，第一层的含义的话是我们的生理健康的状态是非常好的，我们能够有这个听的能力叫做能够听到。第二个含义的话是指我们愿意听，我们能够把注意力放在听这件事情上，能够听到对方的声音，对吧？这个叫做听到以后我们要专注和理解。什么叫专注？有的时候我们虽然表面上好像在听别人说话一样，但是我们却不够专注，所以我们会漏掉很多的相关的信息，甚至有的时候我们只听我们愿意听的，我们想听的，那不够专注我们就会漏掉很多的信息，我们能够接受到的仅有的信息还要去怎么样去理解？那有的时候我们可能会开动脑筋，放在心里去想一想，理解一下，但是绝大部分的时候我们在倾听的时候都是不愿意去理解的，或者是理解的都是不对的，那么理解的不对的话，就会导致信息的偏差，导致我们没有办法更多的更清楚的去了解别人到底想要什么，想表达什么。


在专注和理解的基础之上的话，我们还有一个过程，叫做语言的反应和肢体的反应。语言的反应的话包括两个方面，一方面就是我们刚才讲的打断别人插话、反驳、质疑等等，对吧？这些的话其实都是会阻碍倾听的，阻碍对方的表达的。还有一种语言的反应的话，我们后面也会提到它是各种各样的其他的语言反应，可以来帮助我们引导对方更多的说，更多的去表达，这样的话会有利于我们的倾听，得到更多的信息。同样肢体的反应也是一样的，有些肢体反应可能会让对方不想表达，跟我们产生距离，但是有些肢体反应的话，可能会让对方觉得我们更信任他，也愿意更多的表达。


语言反应和肢体反应都有正向的，都有负向的，它会影响整个这个倾听的过程，在这之后的话就是澄清和确认了，如果我们真的能做到倾听，那我们就一定会有澄清和确认的这样的一个步骤。大家可以仔细的去回想一下，当你很认真的去倾听一个人的时候，你真的很想了解他，很想解决一些问题的时候，你是不是要在他说完以后去对他的某些内容，某些信息做一些澄清和确认，只有澄清和确认你才能确保你听到了他的信息和你理解的信息也是一致的，只有这样的一致的话才能提高我们沟通的效率，才能更好的达到我们沟通的效果，对吧？就像我们在沟通的时候去把沟通分为三个模块一样，我们在这里的话也会把深度倾听的过程分为三个模块，找到这三个模块，并且仔细的去分析它，在这三个模块的过程当中去提升自己的能力，就能达到更好的深度倾听的效果。


提升我们倾听的技巧是哪三个模块？同学们你们也可以想一想，这个过程当中我们可以分为哪三个模块？首先第一个模块的话，我们叫做接收的模块，就是专注和理解我们怎么样能够更加的专注，更加的去理解对方。第二个模块的话我们叫做回应，如何通过语言的反应和肢体的反应做出更好的回应的方式，让对方更好的表达，更愿意多表达。那最后一个就是确认，我们通过确认的方式能够确保我们得到的信息是准确的，和对方是对齐的，是一致的。这三个过程，接收、回应和确认就是我们在这个章节刚开始要说的三二原则了。
