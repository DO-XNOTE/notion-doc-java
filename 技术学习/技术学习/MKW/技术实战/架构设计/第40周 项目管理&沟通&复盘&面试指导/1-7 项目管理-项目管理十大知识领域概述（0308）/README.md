首先左侧是第五版的，右侧是第六版的，上面就是五个过程组，十大知识领域就是左侧第一列，项目整合管理、范围管理、进度管理、成本管理、质量管理、资源管理、沟通管理、风险管理、采购管理和相关方管理。每一个知识领域，在这五个过程组都有自己的内容，比如说整合管理，可以看到在收尾过程组只有整合管理的工作。在启动过程组由整合管理和相关方管理的工作。而在规划过程组可以看到内容是最多的，在监控过程组也是内容最多的，几乎规划和监控是涵盖所有的知识领域的。在执行过程组可以看到范围管理、进度管理和成本管理是没有相关工作的，所以整个 PM book 知识体系这个是最重要的。


一张图，理解了这个项目怎么赚的，怎么管理，什么时候切入，又采取什么比较好的一种方式去进行干预、监督和控制，这张图起到着非常重要的一个作用，所以这一节主要是领大家能有一个宏观的认识这五大过程组合，十大知识领域他们是怎么合作的？组合出来的内容都有哪些？都有什么特点？接下来会重点讲一下十大知识领域里边的一些内容。这一节就先到这里，我是Kelly，我们下一节再见。
