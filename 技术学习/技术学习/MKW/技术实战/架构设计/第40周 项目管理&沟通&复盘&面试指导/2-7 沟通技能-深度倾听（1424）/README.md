# 2-7 沟通技能-深度倾听（1424）

我们除了要了解阻碍倾听的行为之外的话，其实更多的是要去了解无效的倾听，我们需要了解无效的倾听的种类，因为我们只有了解了什么是无效倾听，才有可能去避免无效倾听，才有可能把我们的倾听做得更加的真实，更加的彻底，对吧？那在这里我把这个无效倾听分为三个大类，分别是不想听、不专注和不理解。

第一类叫做不想听，大多的话分为两种情况，第一种情况的话叫做错误的假定，第二种情况的话叫做刻板的印象。比如你有一个同学，他平时学习成绩非常的差，那突然有一天他和你争论一道非常难的数学题，他表达了一些自己对这个题目的看法，以及表达了他如何去解这道题目？由于你平时对他的印象就是学习差，你对他的假设就是他不会做，他去表达自己的观点的时候，你认真去倾听他的概率有多大？我认为这个概率应该是很小的。


所以我们在很多时候会因为我们已经给了一些人一些假定，或者是一些刻板的印象，导致了我们在倾听它的时候就不自觉地带入了这些假定和印象，我们就可能不愿意去倾听了，不愿意去听他讲了。但是往往对方想要表达的深层次的意思和内容，其实是很值得我们去听的，很值得我们去学习的，我们往往就错过了。


第二类的话叫做不专注，不专注又包括虚伪的倾听和选择性的倾听。虚伪的倾听的话其实就是在表面上表现的好像在倾听，甚至在表现的这种虚伪的倾听的表面的话，他很可能还会看着你的眼睛，还会不停的点头，甚至嘴巴里边还会说，对对对，是是是，你说的没错，但是当你说完以后你再去问他。唉，我刚才说的重点是什么呀？他可能完全就不知道，完全就回答不上来。


就好像有一次我跟一个同学去沟通合作，我讲得很清楚，我的这个部门跟他合作需要做哪些事情，目的和意义是什么？我们接下来怎么做？每当我说完一段的时候，这个同学都会说，好的好的，我记住了，并且会很真诚的点头，但是当会议结束以后，什么事情他都记不住，因为他在这个听的过程当中只是假装在听，假装在点头，在跟我附和。但实际上他的心思根本就不再倾听我，那这种倾听的话就叫虚伪的倾听。同学们，你们可以去回忆一下，你在日常的工作当中去倾听别人的时候，你有多少的概率是在做虚伪的倾听？你有什么样的一个场景，或者是什么样的一个情况之下，你是在做虚伪的倾听的。那大部分人的人其实都会有这样的一个感受，或者是这样的一个体验。
那第二个的话叫做选择性的倾听，顾名思义就是只对自己感兴趣和对自己有利益的内容，认真的去倾听其他的内容的话，就拒绝了，耳朵就闭上了。有的时候我们再去看电影的时候会插播一些广告，你会发现插播广告的时候，我们就开始打开手机去看手机了，那这个时候电视传来的这些广告的声音的话，其实我们就已经选择性的屏蔽掉了，就已经听不到了，而且的话会屏蔽得非常的干脆，你根本就感觉不到电视机在响，这个就是选择性的倾听。


在工作当中的话，选择性的倾听的话也非常的多，而且也是非常的严重的。我们在开会的时候，比如说跟产品的同学去开会，当提到一些跟技术实现相关的内容的时候，我们可能就会认真的去听一听，去看对方要说什么，对方的需求是什么。但是当产品同学去说一些业务相关的内容的时候，我们很可能就会把耳朵闭上了，不听了，这个就是一个非常典型的选择性倾听的一个场景，那其实在介绍业务的过程当中，也许我们需要去倾听，需要去了解，可能对我们的技术实现也是有帮助的。


由于我们的选择性倾听屏蔽掉了这些内容，导致了我们沟通效果的一个降低，最后一个的话叫做不理解，不理解当中的话又会包括自恋的倾听和防卫性的倾听。自恋的倾听的话是我们虽然听到对方在说什么，但是不会的认真去理解对方所说的内容，而且总是会把这个话题不自觉地转移到自己的身上。例如有一个同学跟我们去沟通，他表达的内容是他如何去对一个新的系统进行搭建，还没有等他说完，我们就把这个话题转移到自己的身上了，开始跟他去描述我们是怎么做的，我们在这个新系统的过程当中遇到了什么样的一些问题，我们是怎样完美的解决的？以此想要表现自己的实力和自己的能力，那完全不顾对方说这件事情的目的和他背后要表达的是什么。


这种倾听的话就叫自恋的倾听，那自恋的倾听的话更多的会出现在孔雀型的人的身上，他是完全不在乎别人的，不去关注别人的，只在乎自己，只想去表现自己。最后一种的话叫做防卫性的倾听，防卫性的倾听的话就是说我们在倾听的时候，总觉得别人的说话的内容和信息是在攻击自己，或者是在给自己挖坑，有一种总有刁民想害朕的这样的一种臆想和假想，随后我们在完全不了解对方说什么，那样的情况下就开始反击，就开始防卫。这样的一个倾听的过程就叫防卫性的倾听，不在乎别人说什么，我只要怼你，我只要去反驳你，这个就是防卫性的倾听了。


防卫性的倾听不仅会阻碍倾听，不仅会让倾听是无效的，得不到有效的信息，阻碍双方的沟通，阻碍沟通的氛围，还会导致双方的关系越来越远，甚至会产生一些矛盾和冲突。所以防卫性的倾听的话也是大家要注意，千万不能出现的。好了，说了这么多阻碍倾听的行为和无效的倾听内容以后，我们总结一下以上的内容的话，其实都会从几个层面来去进行分析，一个层面的话叫做行为阻碍，一个层面的话叫做心理阻碍，刚才的不能过早的出现的阻碍倾听的行为，比如说询问、比如说批判、比如说建议、比如说安慰，这些更多的是属于行为层面的阻碍。


我们这个页面讲的不想听、不专注和不理解的话，更多的是属于哪里是属于心理层面的阻碍，不管是行为层面还是心理层面，都会去阻碍我们的倾听，让我们的倾听变得无效，所以在这两个层面的话，大家是非常非常要注意的。那最后一个叫做环境的阻碍，也就是可能我们在倾听别人的时候，由于外界的这个噪音很多，导致了我们听不到，听不清楚，这个的话大家只要去注意，如果你有非常重要的需要去倾听的对象的话，找一个安静的地方，找一个安静的会议室就可以了。如果提前出现的行为会对亲情造成的阻碍，以及无效的倾听的这样的一个内容，大家能够非常好的理解，非常好的去吸收的话，那么恭喜你，你其实在倾听的这个过程当中已经进步很大了，因为我们还记得刚才讲的这个倾听的流程吗？第一步就是什么呀？就是听到啊，如果听到都听不到，那就不谈后面的流程了。


那么阻碍听到的这个环节其实就是我们刚才讲的行为的阻碍和心理的阻碍，所以大家一定要去反思自己有没有出现一些阻碍倾听的行为，以及无效倾听的这样一些状态。



可是我们也去想一想，假如我们面对一个人，无论你说什么，对方都无动于衷，他没有给你任何的回应，也没有动作，也没有表情。请问你还有继续说下去的欲望吗？我想大多数的同学可能都没有了，对吧？所以在倾听的过程当中，我们的第二个原则叫做回应，也非常的重要。

回应的话，它指的是我们用肢体或者是语言向对方传递，我们非常投入到倾听的这样一个状态。回应的话主要包括肢体性的反应。肢体性的反应包括哪些？同学们可以想一想吗？能够去说出几个来吗？

比如说点头、微笑、目光接触、肢体同步和身体前倾，这些都是肢体性的反应。


同学们，如果你们有非常的深入去倾听别人的经历的时候，我想你们应该对于点头微笑和目光接触都是有感受的，因为当我们去认真听别人的时候，可能会不自觉的就会点头了，不自觉的会跟对方做一些目光的接触，甚至的话我们的身体都会前倾，这些肢体性的反应有什么意义？它能带来什么呢？首先的话它能让对方觉得我们非常的友好，对吧？而且我们在对方表达的时候，我们是注意力非常的集中的，点头目光接触的话还会给对方一种暗示，就是我们对他说的内容的话是非常的认可的，而且我们也是非常的感兴趣的，这样的一个过程会让对方非常的有自信心，而且会激起对方非常强的一个表达欲望。


我们还记得在沟通的高手的那个章节，我们讲过沟通的高手的一些特质，其中一个就叫做多样的行为反应，那么这个肢体性的反应其实就是多样的行为反应当中的一部分，我们用这些行为反应的话，其实能够做更好的深度倾听，能够更好的了解别人。那么扩大的是我们乔哈里视窗的哪个？就是我们的盲区，对吧？让对方讲的更多，让对方表达的更多，所以扩大我们的盲区，了解对方了解的更多，更好的能够达到我们沟通的效率，达到我们沟通的目标，对吧？

这个复述的过程会有助于我们理解对方要表达的意思，并且可以和对方做一个简单的澄清，让对方看一看我们复述的内容和他要表达的内容是不是一致的。那如果对方听到我们复述的内容是错的，或者是有问题的，他可能会怎么样？再解释一下，这是有助于双方对于沟通的过程当中的信息交换的一个确认和对齐。


而赞美就不用多讲了，我们通过赞美的方式可以让对方觉得我们很认可他，我们很关心他，我们很关注他，我们的兴趣是很浓烈的，对方的表达欲望就会很强，尤其是在对方表达自己的一些成功或者是自己的一些沮丧的这样的一些行为的时候，我们恰当的使用这种赞美和认可可以缓解对方的一些情绪，可以有利于建立更好的这样的一个沟通的氛围。


所以在倾听的过程当中，我们也说了这个对方表达内容背后的那些情绪和情感，是我们时时刻刻要关注的，关注到以后我们可以用语言性的反应去做一些回应。那么后面我们讲的是澄清和认可。认可其实跟负荷负数赞美其实有一些交界的地方，但是认可的话有的时候也就是一种非常简单的直白的一种认可，也可以给对方一些信心，也可以告诉对方我们对他的一种感受，是非常的相信的，非常信任的这样的一个状态。那澄清是什么呢？大家还记得表达的语义规则当中的问题吗？我们讲了哪些？比如说模棱两可相对语言，当我们听到对方在表达的时候，出现了一些模棱两可相对语言语义的时候，我们该怎么办？这个时候我们最好就是用澄清这样的一个方式去解决这个问题。那假如你的朋友告诉你，他可能要迟到一会儿，那这可就麻烦了，因为什么呢？我们学过了，可能要迟到一会儿，可能就是一个相对语言，对吧？一会儿也是一个相对语言，这个时候他到底还来不来？如果他来的话，他要迟到多长时间？这些都是不确定性的，这个时候我们可以跟他做一个澄清。


我们怎么说？你的意思是你会迟到 10 分钟，但是你一定会来，对吗？那么对方可能就会对这个问题进行详细的解答，我们双方交换的信息就会更加的准确，更加的明晰，对于沟通的效率的提升也会更加有帮助，对吧？澄清其实它是需要一些技巧的，澄清的关键是在什么地方澄清的关键的话是我们首先能够认真的听到对方表达的内容，其次的话我们对于对方表达的内容的话，要有一个自己的简单的理解和认知，在这个理解和认知的前提之下，我们用自己的理解来复述和转化这些内容，并且把这些内容表达出来询问对方，然后让对方根据我们的询问了解我们是否真正的听懂了他要表达的内容是什么。所以澄清的话它是需要渐进的过程的，可能我们需要仔细的去听，听完以后再去做这件事情。
