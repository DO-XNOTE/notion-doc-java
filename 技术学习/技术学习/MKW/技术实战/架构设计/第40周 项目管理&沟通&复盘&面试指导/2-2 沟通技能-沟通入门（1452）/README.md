早期的时候沟通被认为是一门社会科学，而且社会科学家为了完整的去描述沟通的过程以及模式，他们把信息传递画了一张图，试图通过这个图来反映沟通的过程。最开始这个图叫做线性的沟通模式图，这个图长什么样？我们一起来看一看。在沟通的过程当中，一定会有一个传送信息的人，对吧？我们姑且把它叫做传送方，那么相应的也会有一个接收信息的人，我们把这个人叫做接收方，而传送方和接收方之间存在的那个就叫做信息。所以线性的沟通模式图当中，传送方、信息和接收方被称为是三大要素。那么同学们可以思考一个问题，在信息通过传送方传递给接收方的这个过程当中，会不会出现相关的损耗而降低沟通的效率？我们说是会的，那么信息会在哪些过程当中出现损耗，来降低沟通的效率？首先我们想一下传送方要把自己的想法传递给接收方的第一步是什么呀？没错，就是要把自己脑袋里边的想法编码成可以传递的信息，对吧？比如我们把自己的想法，把自己的思想变成可以传递的声音，变成可以传递的文字，那么我们把自己的想法变成可以传递的文字和声音的这个过程会出现信息的损耗。


因为很多时候我们都会有一种经历，就是明明我有一个非常好的想法，但是我把这个想法说出来或者是写出来的过程当中，会发现没有办法完完整整的，非常细致的把这些东西表达出来，那其实就是我们在编码自己的想法的过程当中出现了信息的损耗，那么同样接收方在接收到传送方传来的信息的时候也要进行解码，他需要把这些文字、声音通过大脑的解码变成自己可以理解的内容，这个过程也会有信息的损耗。


比如传送方传过来的信息里面包含了三层的意思，那么接收方根据自己的解码只解码了两层的意思，那么另外一层的意思其实就是信息的损耗了。所以在整个的线性沟通模式的传递过程当中，信息在编码和解码的过程当中都会有损耗。好了，我们再想一想，信息除了在编码和解码的过程当中有损耗，在传递的过程当中那些渠道的过程当中会不会有损耗？我们说也是会有损耗的，那这个损耗我们叫做噪音，主要分为三大类。第一大类叫做环境的噪音，比如说我现在正在给大家讲课，那么突然我隔壁的房间开始装修了，咚咚咚，当当当当，那么这些声音可能会影响同学们听清楚我讲课的内容，那这个噪音我们叫做外在的环境噪音。还有一种噪音是由于我们在沟通的过程当中身体不适，导致了我们没有办法集中注意力，或者是我们没有办法很好的进行逻辑思维的思考，表达自己的内容，那这些噪音叫做生理的噪音。第三种噪音叫做心理的噪音，是由于在沟通的过程当中，心态的原因导致不想听，不愿意听，或者是不想表达，或者是表达的不够具体，而造成的沟通效率的降低。好了，这个图目前就是线性沟通模式的全貌图了，请问各位同学们，你们认为这个图可以完整的展示沟通的过程吗？

不过这个案例是非常简单的，它主要讲的就是一个年轻的产品负责人跑到一个研发负责人强哥那边去控诉研发的系统崩溃了，导致了业务的这样的一个影响。


而且绵阳还被强哥的一些不太精神饱满的状态给激怒了，跟强哥沟通的过程当中非常非常的生硬，那强哥作为一个老研发，没有和这个年轻的产品经理一般见识，他建议这个产品经理坐下来慢慢说，把自己的一些相关的问题表达描述清楚。那绵阳在这个过程当中发现强哥也熬夜了，也在加班加点去做一些技术的需求，产品的需求也非常的辛苦，所以就调整了自己的这样的一个沟通的模式和状态，最后两个人共同的把一次故障给解决了。这个案例其实是产品和研发同学在日常的工作当中经常出现的一个沟通场景，在后面的课程内容我们还会多次的用到这个案例，所以同学们可以稍微的暂停一分钟，来仔细的阅读一下这个案例。


通过这个案例我们再回过头看刚才的沟通模式的定义图，我们会不会发现一些问题？首先第一个问题，在这个案例当中，谁是传送方，谁是接收方？我们会发现在这个沟通的案例当中，好像绵阳和强哥都在时时刻刻的相互作者信息的传送和接收的工作，比如说绵羊气喘吁吁地去表达了系统崩溃的问题，而强哥也用看了看，撇撇嘴，挥挥手这样的方式来给绵阳传递他的相关的信息。


所以在整个过程当中，我们很难的明确的去区分谁是那个单纯的信息传送方，谁是那个单纯的信息接收方。因此我们更希望在沟通的过程当中，把所有参与沟通的人都叫做沟通者，而不叫做去区分传送方和接收方了。另外很值得注意的是，我们在这个案例当中信息交换的方式和线性沟通的这个模式图有什么样的区别？首先第一个区别是信息已经不再仅仅是语言或者是文字了，沟通者的状态、行为、表情、语气、音调，甚至是穿衣打扮，其实在沟通的过程当中都在传递着相关的信息。比如我们在第一个自然段，我们通过第一个自然段的描述可以体会到绵羊的心情是什么呀？很焦急，很急躁，对吧？我们是怎么发现绵羊焦急急躁的？我们会通过几个词来发现绵羊很焦急、很急躁，比如说什么呢？气喘吁吁，还没站稳，开始大喊，对吧？这些词其实它描述的是绵羊的一个状态，一些表情、动作和音调，那这些信息其实能够告诉我们绵阳是一个什么样的状态，这就说明了这些非语言的信息有的时候比语言信息更能让我们在意，更能帮助我们了解沟通者他要表达的一个状态和内容。


其实沟通学家在很久以前就做过一个实验，他们让沟通者在沟通的过程当中，故意的去做一些比较夸张的表情和比较夸张的动作。过了很长一段时间以后，他们再去采访当时沟通的那些人，他们问，当时你还记得在沟通的过程当中对方说了什么吗？大部分的人都已经记不住了，但是大多数人却能记住当时对方做的那些夸张的表情和动作。


这也就验证了在沟通的过程当中，其实非语言的表情状态，这些音调、眼神可能会更让我们在意，也更能帮助我们去了解对方的状态。但是反过来我们自己在沟通的过程当中，也要时刻注意自己要去避免一些多余的表情、动作、神态、语音和语调，因为很多时候这些传递给对方会让对方去解读，有可能会解读出来一些负面的这样一个信息，导致沟通的障碍，甚至会发生一些冲突。所以想要提高沟通的效率，想要让我们把沟通做得更好，我们不仅要注意自己的语言的表达，还要注意自己的非语言信息的传达。我们再来仔细的看一下这个沟通的案例，我们会发现在这个沟通的案例当中，信息传递会出现非常大的损耗，导致了沟通效率的降低。那么信息的损耗会受到哪些因素的影响？我们刚才已经提到了，比如说生理方面的影响，比如什么呢？我们看强哥他怎么样通宵了，通宵以后身体的状态一定是不好的，所以第二天早上产品负责人绵阳过来去沟通的时候，这个强哥因为熬夜了，脑袋瓜子嗡嗡的，所以这个时候他的沟通的状态一定会受到影响，这是生理方面的影响。而绵阳它其实是一种心理方面的影响，因为他的心态是很着急的，是很焦虑的，所以也影响了他的这样的一个沟通的效率。


那除了生理影响和心理影响造成的沟通的这个效率的降低，还有什么造成了沟通效率的降低？我们会发现，其实绵羊和这个强哥他们是有不同的经历的，至少他们的专业知识和他们的工作目标是不一样的，对吧？一个是研发的这个逻辑思维，一个是产品的逻辑思维，不同的逻辑思维对于事物的理解、表达和信息的解码其实都是不一样的。


所以在日常的工作当中，研发和产品在沟通的时候，总是会不自觉地站在自己的角度去发表意见，并且他们也会发现自己说的话对方为什么就是听不懂。其实这也是产品和技术在沟通的时候容易出现冲突的原因之一，因为谁都不理解对方到底在表达什么，因为他们会有不同的背景，不同的知识，不同的思维逻辑，对吧？另外我们再来看一下在这个沟通的案例当中，信息传递的渠道包括哪些？除了有生硬的渠道，还有什么呢？来看一下这个地方视频案例，那视频案例其实也是一个信息传递的渠道，还有比如说通知的短信也是一个信息传递的渠道，这说明沟通的过程当中，信息传递的渠道是多种多样的，不一定非要通过声音、语言或者是文字，而且不同的信息传递渠道对于沟通的效率的影响是非常大的。比如说这个视频案例，产品同学如果去跟研发同学描述一个故障的过程当中，他会描述的可能不太清楚，因为他自己对于技术的这种系统，这种逻辑理解是不够深刻的，甚至他的专业知识也是不足的。


但是如果他能把这个故障录成一个视频，把视频直接给研发同学看，那这个信息一定是更加完整的，也是更加有效的，还会有很多的例子去证明信息传递的渠道的不同，会导致我们信息传递效率的这样的一个不同。比如说我想向一个心目中的女神表白，把我爱你这样的一个信息传递给他。我有三种不同的传递渠道，第一种我给他发一个微信，告诉他我爱你。第二种我捧着一束鲜花走到他的面前，跟他说我爱你。第三种是我把他办公大楼的外表面包下来，在上面用灯管连接成我爱你的字样，当他晚上下班的时候，我带他去大楼外面看这个字样写的我爱你。请问各位同学，哪一种信息传递的方式和渠道会让这个女神更加的震撼？我想一定是第三种，所以你看相同的信息用不同的渠道去传递，所得到的结果，甚至是沟通的结果和沟通的效率其实都是不一样的。因此我们在未来的沟通的过程当中，不仅要去思考自己要传递什么，也要去思考我们用什么样的沟通渠道，信息传递渠道更能达到我们的目标。


接下来是最后一个问题，请问各位同学，这个沟通的案例是在什么时候结束的？可能很多同学会说，那就是在绵阳和强哥说完话以后，他回到了自己的工位上，那这个就结束了。比如说让绵阳回去检测活动数据的时候，绵阳回去了，沟通就结束了，真的是这样吗？其实这往往是沟通的一个非常大的误区，其实我们会认为当绵羊收到了问题解决的通知短信的时候，这个沟通才是真正的结束了，


大家一定要记住，评判一段沟通是不是完整的结束的，这个标准是沟通有没有达成共识并且达成的，这个共识要坚决的执行到位，并且给了相应的反馈了，这个沟通才是结束的。


如果我们发现一段沟通仅仅是达成了共识，但是这个共识却没有被落实到位，那其实这个沟通是怎么样的？它就是一个无效的沟通，因为沟通的共识没有落地执行，所以沟通一定是达成共识且执行落地到位，并且有了相应的反馈才是完整的，这是非常重要的，大家一定要记住。
