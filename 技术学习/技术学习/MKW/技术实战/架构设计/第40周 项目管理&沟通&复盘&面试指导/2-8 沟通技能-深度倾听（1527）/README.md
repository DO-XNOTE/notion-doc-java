让我负责剩下的一部分。结果工作出现了问题，是出在他做的那个部分，但是领导就因为我是最后签字的人，就把我给批评了。


那我就问他，你哭的原因是什么呢？小木说，这个错误明明不是我犯的，领导凭什么骂我，而且还让我写书面检查，夹在我的档案里，肯定会影响我的实习分数，关系着我能不能留在公司。那我就问他，那你解释没有？小木说解释了，可是领导根本不听，而且领导也知道任务是同事的，说的错误跟我没关系，而且同事明知道我挨批了，都没有帮我解释，都赖他，我怎么办好了？在这个案例当中，我请问各位同学，你们都听到了什么？有的同学可能会说，我听到了小木的委屈，因为小木一直在哭对吧？一边哭一边说，也有的同学说，我听到了小木的抱怨和愤怒。


那么小木找元宝来倾诉，真的是为了表达委屈、抱怨或者是愤怒吗？请问各位，小木在这一段话里边的哪一个地方，明确地、清晰地表达了自己的委屈、抱怨或者是愤怒？哪一句话，哪一个词找不到没有？那么我们来看一下，小木都说了什么呢？我们凭什么来判断小木是在表达抱怨、委屈和愤怒？第一段话小木说的是同事生病了，自己帮别人完成工作，对吧？工作出了问题，是对方做的那个部分，他是最后一个签字的人，他被批了。


第二段话，小木表达的是什么呢？让他写了书面的检查，影响他的实习分数，关系着他能不能留在公司，对吧？最后一段，小木表达的是解释了领导不听，同事没有帮他解释，对吧？那么我以上讲的这些都是什么？都是事实，如果小木没有说假话的话，这些表达的都是事实，如果这些都是事实，这些是可以被我们直接拿来怎么样使用和分析的信息，而这些信息我们分析出来的是什么呢？就是有的同学分析的抱怨、委屈、愤怒，抱怨、委屈和愤怒是我们根据这些事实怎样猜测出来的？我们分析出来的，我们猜测的和我们分析的它一定就是真的吗？不一定。而且我们猜测的和分析的是小木的什么呀？委屈、抱怨、愤怒都是什么呀？都是小木的情绪。所以我们在倾听的时候一定要区分清楚哪些是我们听到的事实，哪些是这个事实背后的一些情绪，而事实的话是可以直接相近的，直接使用的。


但是那些分析出来的情绪的话是怎么样？却不一定是真实的，除了事实和情绪之外，这个里面还有什么？还有小木说都赖他，对吧？都赖他，这个是什么呀？这是一个观点，这是小木的观点，这个东西它更多的是主观性的东西。一定是事实吗？一定是对的吗？也不一定。除了观点以外还有什么让我斜书面检查我到底能不能留在公司，所以我们会猜测小木他还有一个需求，他的需求是什么？是想留在公司对不对？所以通过我们刚才以上的分析的话，

事实、观点、情绪和需求是我们在听到了对方大量的信息的之后，我们需要去做区分，需要去做拆解的这样的一个内容。那么事实是什么呢？事实是对方表达的实际情况是已经发生的事情，可能有一些数据，对吧？过去发生的一些人的行为，对吧？这些都叫事实。当我们听到一些事实的话，对方表达的可能不是具体的，可能不是非常清晰的。我们如果想了解这个事实的背后的真相和深挖的话，我们要把这个事实具体化，具体化的过程的话，就是问了一些相关的问题。


那第二个的话，我们可能会通过对方的表达的信心得到一些对方的观点，那比如说都赖他，那这个观点的话，其实我们刚才也讲到了在第二章的内容，观点是一种标签化的评价，有可能是吧？那么这种标签化的评价它本身是没有什么太大的意义的，对吧？他可能是错的，可能是假的，那这个观点它背后得到这个观点的依据的那些事实，依据的那些行为，依据的那些数据，其实对于我们来讲可能是更有用的。


所以当我们听到别人的观点的时候，当我们猜测到别人观点的时候，我们要去根据这个观点去挖他背后的那些依据，那些依据的话有可能是其他的一些相关事实，可能是我们能够使用的信息。那第三个的话叫做情绪，比如说我们猜测小木的情绪是愤怒、委屈、抱怨，如果真的有这些情绪的话，我们在整个沟通的过程当中，其实是应该先去解决情绪，只有把情绪解决了，我们才能真正的去理性的思考，理性的对话。很简单的例子，如果各位同学你们有女朋友的话，当你的女朋友特别生气的时候，你给他讲道理，你的女朋友会怎么说？到底是你的那些道理重要还是我重要，对不对？你不解决他生气的这个情绪，你给他讲道理是讲不通的。所以我们在职场当中去倾听别人也是一样的，我们倾听了对方的情绪以后，如果这个情绪非常的大，非常的负面，我们也是一定要去先解决情绪，再解决事情，再去讲道理。


那最后一个就是需求，如果我们通过倾听，通过信息发现对方有一些需求的话，我们一定要对需求去做澄清和确认。比如说刚才的这个案例，小木哭着找到了，我就说了这么多，也许他的需求的话是想留在这个公司，也许他的需求的话是想得到我的安慰，但是如果我没有跟他确认，我就直接安慰他了，我说的特别特别多，安慰他的话会发现怎么样没有用，安慰了很多很多，他还是不开心，他还是很担心他还是很焦虑。原因是什么呢？原因是因为他来找我寻求安慰，根本就不是他真正的需求。他真正的需求可能是什么呢？可能是他想找到一个继续留在这个公司实习的这样的一种方法。如果我跟他去确认一下，知道了他真正的需求，并且帮助他解决了留在公司的这样一个问题，他可能才会真正的怎么样走出这件事情的阴影，对吧？所以需求的确认也是非常的重要的，通过我刚才对于倾听地图的介绍的话，

你的意思是说想怎么怎么样是吗？我理解到的是什么？什么什么是吗？你告诉我的意思是什么？什么什么对吗？我听到你说什么什么什么，对吗？总的来说就是什么什么什么是吗？所以你的想法是什么？什么什么是吗？对吧？这些都是比较简单的一些语义的规则，大家可以直接的去使用。但是有一些地方，有一些场景是不需要做确认的，比如说对方表达的内容已经非常的具体了，表达的信息的话我们已经完全理解了，这个时候不需要做确认。


我举一个简单的例子，公司来了一位新的同事，他向我走了过来，然后跟我说，同学，前面左转那边是不是洗手间？你能不能告诉我一下？那这个时候我说，唉，同学，看起来您是憋不住了是吧？这个确认其实就是没有必要的确认了，对不对？所以确认指的是在倾听对方的一段话之后，做一个总结提炼，防止出现错误。所以我们要去跟对方做确认，当我们能够准确的理解对方的含义，当对方表达的含义足够的清晰的时候，确认就可以省略了。确认是一个比较简单的事情，难的话是难在大家可能会忘记做确认，所以我们在倾听的最后，只要有意识的能够记得要去做确认就可以了。

首先的话，我们讲倾听的三耳原则，分别是接收、回应和确认。其次我们讲了倾听的阻碍，分别是环境阻碍、行为阻碍和心态阻碍。具体来说的话，行为阻碍包括询问、安慰、批判，或者是给出相关的建议，对吧？那心态阻碍的话包括三大类，不想听、不专注和不理解。最后我们讲了倾听的地图，里面包括了四个非常重要的相关的要素，就是事实、观点、情绪和需求。我们在倾听别人的过程当中，一定要注意去区分哪些是事实，哪些是观点，哪些是情绪，哪哪些是需求。如果我们听到的事实不够具体，我们要把事实具体化，如果我们听到了对方的观点，就要去挖掘这个观点背后的依据，找到相关的事实和行为。如果我们了解了对方的情绪，先解决他的情绪，再做理性的沟通。最后我们如果感受到了对方的需求，需要跟他去做一个需求的确认，再通过沟通的方式看能不能满足他的需求，更加的了解他，达到我们沟通的目标和目的。以上的内容的话，希望对大家在倾听的这个过程当中有所帮助。
