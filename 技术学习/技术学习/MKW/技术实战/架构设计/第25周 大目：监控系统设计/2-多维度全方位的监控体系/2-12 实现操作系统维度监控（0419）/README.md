大家好，我是大木。这节课我们来用 node exporter 去实现操作系统维度的监控。有关 node exporter 我也写了一篇，手机一起来看一下。首先到这个地址下载 node exporter，同学们可以根据自己的操作系统去下载，

[https://github.com/prometheus/node_exporter/releases/tag/v1.5.0](https://github.com/prometheus/node_exporter/releases/tag/v1.5.0)

对于网络不好的同学可以到这里去下载。大木已经把常用操作系统的 node exporter 上传到了这里。


OK 下载完成之后解压。 **command o**。解压完成之后，打开一个新的窗口，并将目录切换到 node exporter 的根目录，再执行点杠 node exporter，这样就可以启动 node expoter 了。启动之后访问 localhost:9100/metrics，可以看到当前操作系统的各种健康指标。
