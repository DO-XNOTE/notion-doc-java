一是瞬时向量。所谓瞬时向量，表示某个瞬间的数据，比方在某个瞬间 CPU 的占用率。

第二是区间向量。它表示一段时间的数据，一般是多个瞬时向量的组合，比方在 5 分钟以内的 CPU 占用率，它其实是由多个瞬时向量组成的集合。
