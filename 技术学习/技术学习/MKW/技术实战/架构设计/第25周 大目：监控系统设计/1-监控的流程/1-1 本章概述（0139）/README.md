hi，小伙伴们大家好，下面进入监控系统这一阶段的学习。这张叫做监控系统概述，它是一个偏理论的章节，主要探讨这么 4 个话题一是监控系统常见的目标，也就是监控系统应该具备哪些能力。二是常用的监控维度，监控的范围和方向应该有哪些。三是监控的常见流程以及方法，指的是监控系统从开始到结束的流程，以及监控系统设计时候的常用套路。四是常用的监控指标以及告警设计时候需要考虑的一些诉求。总的来说，这章的内容稍微有点散，但其实都是在围绕监控系统需要有什么这样的一个话题来探讨的。

顺便说一下，这段的内容可能会稍微有点枯燥，事实上，在咱们这一阶段采用的是先理论后实战的模式进行讲解的，之所以这样设计，主要是考虑监控这个玩意儿，虽然可能大家都在使用，但涉及到的东西还是比较多的。它有一套非常庞大的知识体系，如果采用理论和实战穿插的模式，很难把理论的内容都覆盖到了，所以做了这样的一个设计。但是这样一来，也导致了这一阶段的内容会相对慢热一点，对于不爱听理论的同学可能不是很友好。所以如果同学们比较偏实战，建议可以直接跳到后面的实战章节，照着实战再回过头来看理论，这样也是 OK 的。下面我们正式进入本阶段内容的学习。
