上面是这样定义软件系统架构的。软件系统架构就是描述计算组件和组件的交互。所谓组件是静态的，就是一个一个的模块，一个一个的领域，一个一个的包，它的交互是动态的，这就是中间的逻辑过程，它产生了实际的计算机系统对不对？那这都是组成派的铁证，那如果是三遍出来了，那就更加完美了。
