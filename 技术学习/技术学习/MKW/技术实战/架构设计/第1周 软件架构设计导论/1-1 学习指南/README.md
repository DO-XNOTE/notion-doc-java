一期的架构师成长直通车我们主要是围绕在应用技术层面，那突出的是架构之术。那什么是架构之术？那这就是指硬核的技术，那就像我们抗战剧里的雷人镜头之一，八百里外一枪干掉鬼子的司令官，这就是过硬的个人技术啊。那我们二期的十项全能课程则会体现出架构之道啊。什么是道呢？咱还拿前面神枪手的例子来说，就是八百里外这么多人，你要知道干掉哪个目标最有利，什么时机动手最合适，这个就是架构之道，简单来说就是如何来用最合适的技术方案来解决业务问题，如何从架构层面来拆解问题，分析问题，防范同类问题的发生，在这个过程当中总结沉淀下来的架构理念。
