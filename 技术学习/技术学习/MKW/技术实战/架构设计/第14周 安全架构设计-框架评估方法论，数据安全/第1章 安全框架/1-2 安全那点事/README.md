那同样我们什么所有架构师的课程都会由粗到细，由浅入深？那前面聊了这 3 信没有进一步展开，那我们就来展开一下机密性其实包含哪些啊？包含用户的身份安全相关的什么逻辑访问控制，还有还有物理安全相关的物理访问控制，以及通过网络安全手段进行什么加密传输和数据安全手段实现的数据视图安全，那同样可用性也是什么？通过我们五芒星五大维度的多个角度来进行展现，比如像数据层我们有数据高可用和容灾来实现什么数据的可用性。
