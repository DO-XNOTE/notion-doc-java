也就是说我先要去识别哪些东西，识别所有的信息资产基础包含什么信息分级追溯、敏感标志，以及需要保持了什么信息安全目标，比如你的安全的合规目标、风控目标等等。了解了你的基础信息，了解你的安全目标，然后我们才能开展保护。那这个保护从信息技术角度来看，分为采集、处理、存储、传输和清除的具体的很多的保护套路。然后从环境上又分为收集、使用、保存、共享和销毁很多很多的保护的方法，这些保护方法都会在后面，什么我们的这个安全无芒星的五个维度，逐一跟大家来聊一聊哪些保护方法。
