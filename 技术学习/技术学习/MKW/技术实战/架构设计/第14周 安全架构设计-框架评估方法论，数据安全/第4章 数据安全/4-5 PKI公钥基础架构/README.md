但回忆想当年能成为老司机之前，是不是有一个考驾照成为新司机的过程？一样的道理，我们要有证书是吧？我们要有公钥。其实我们也会需要有一个什么拿驾照、拿证书的过程，那这个过程通常是先去驾校学习怎么样开车、倒车，我们的这个停车，然后换油门、换挡这些基本功能，在驾校里面好好学习我们这个学习方法其实在证书里面是什么？是通过一些工具获得命令，帮我们创建出 CSR c s r 就是 certificate signing request，就是什么签证书的请求，要生成这样一个请求，然后这个请求发给谁呢？发给交警同志。


交警同志会什么？会作为这个 r a 注册授权人， registRAtion authority 是个权威的注册的这个人员或者机构，他会给我们来个现场路考一考你这个路边停车单向过程当中的这个调头，以及一些基本的这个红绿灯，上高架、上高速的基本能力考核，通过了，它不是现场发证，它是会什么通过全国的指定的那几家什么驾照中心进行这个证书的颁发，这个驾照中心，我们在这个证书行业里面叫什么？叫 certificate authority 证书中心，或者叫颁证中心，这个中心全球有那么几家比较大的，比如像Interest、Verizon、each， Cert 这些都是比较大的全球的企业中心。


他给大家发证书证明什么，你这个企业什么是合法合理的上岗的？你有自己的独有的证书或者证明你个人是在 CA 中心注册过的，给你发一个证书，这时候长什么样子呢？就长这样子，这个小本本里面记录了很多信息，我们后面一个胶片会看一看这证书里面通常记录哪些信息。当你有了本本以后你就可以了，是可以开车了，对吧？可以横行无阻。但只要本本没有过期，你就可以正常的提供网站服务，别人看到你的公钥证书都会信任你，而不用去改变它自己特殊的一些这个，我们的这个 property 文件不用去改变一些特殊的 CS 文件等等都不用去改动。


如果哪一天你酒后违章怎么办呢？会有专门的这个公安机构来吊销你的证书，一旦你的证书被吊销，你就会什么被加到这个叫证书撤销列表， certificate retrieval list，这样一个撤销列表把你加进去以后，如果浏览器或者是一些第三方的客户端在检查你证书的时候，它会通过网络进行一个在线检查，通过比如像 OCSP 在线检查工具或者机构来进行检查。这个时候他会查到一个结果，就是你的证书已经吊销，你不是一个合法的驾驶人了，所以你在开车，是吧？那你就会被什么抓到监狱里面处理，而不是能正常的什么去跟其他的企业进行正常的沟通了。
