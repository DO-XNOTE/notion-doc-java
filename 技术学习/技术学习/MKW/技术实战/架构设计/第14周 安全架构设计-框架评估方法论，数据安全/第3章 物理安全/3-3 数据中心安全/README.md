防止内部的东西遗失。怎么样防止呢？第一种思路就是震慑和阻拦，可以通过比如像这种栅栏或者是铁丝网，防止外面的非法人员入侵或者攀越。另外有很多的警示装置，这些警示装置是什么？既是一个告知，也是一个威慑，还有比如像保安、警犬，这些都是威慑和阻挡的方法。那除了阻挡以外，安防还有一个思路就是延迟，比如说这把锁，虽然锁你可以撬开，但是它是不是可以阻拦你一段时间？除此以外还有一些隔离墙以及一些门，这些门可能是转门，可能是多道转门，甚至是什么多道需要通过指纹识别的这种密码门。通过这样的门的方式可以阻拦或者延缓你什么攻入数据中心的可能性，那除此以外更关键的内容就是检测了我们通过这种摄像头的 CCTV 摄像头来检测所有用户的行为。同时我们还有很多的传感器，比如说声波传感器、红外传感器等等，还有什么感知有没有非法的物体？不管是人、动物或者是其他的各种各样的小物体进入到数据中心里面，最后大部分的这个感测以及感知系统都会把数据对接到后台的监控平台，这样我们的安保人员可以实时监控整个数据中心的运作。


总体来看，其实上面两个不管是威慑还是延迟，属于数据机房的访问控制。下面的检测就是数据机房的入侵检测。诶，这跟网络里面两个概念，访问控制和入侵检测什么是几乎是一致的，但它不是逻辑上的，而是物理上的实现访问控制和入侵检测好聊，再多理论不如实际。下一章节飞扬老师给大家分享一下我曾经参与过的一些项目，其中包含很多的数据中心。薇娅老师曾经拜访过很多数据中心，我们来给大家分享一下什么是安全的数据中心，什么是数据中心？最基本的安防常识，大家敬请期待。
