一个是人员的安全，另外是数据中心其中的访问控制和入侵检测的安全。为什么要把物理安全作为五芒星的第一个来讲呢？因为有这样一句话叫人的安全才是最重要的安全。不管是后面的聊到了通信、数据、软件或者是其他的所有的安全内容都要围绕人的安全，当什么发生灾难的时候，应该先救网络，还是应该先救数据，还是应该先救人啊？当然是要救人为准，所以安全所有的文档，所有的课程里面通常都会把人的安全作为第一内容。


首先来跟大家介绍，辅导老师也不例外，我们会把人和数据中心的安全作为物理安全这样一个整个大章节来首先开讲，但这个大章节其实并不大，我们会以什么？会以一些理论介绍和小故事分享，通过几个小节快速地带大家走完物理安全的内容。因为物理安全其实离我们 Java 工程师或者是 Java 架构师还蛮远的，让我们感受一下什么是物理安全吧。
