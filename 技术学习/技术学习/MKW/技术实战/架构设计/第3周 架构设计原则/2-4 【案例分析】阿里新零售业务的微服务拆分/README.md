比如像这里我们首先第一个业务方就是咱的终端用户，那这些终端用户就是我们俗称的买家，对不对？买家这里有很多入口，比如说你的 APB 终端、购物终端，还有你的手淘手机、淘宝，这里是有窗口来直接导流的，这也是一个流量非常大的入口。还有你的线下，因为新零售讲究线上线下同时打，这个时候你的线下的卖场它有 POS 机，这也是一个用户的一线入口。

那首先有请各位嘉宾先入座淘系中台业务，在我们的商品中心以及整个新零售业务的微孵化过程当中，我们借助了很多淘系中台的业务能力来构建我们的业务系统。咱前面说过中台是什么？中台就是集团层面的一个微服务治理，那淘系的中台为你的业务方提供了低成本的接入，你可以经过很小的业务改造，就将你的整个业务玩法集成到淘系的上下游链路当中。比如说这里有 IC 系统，还有订单系统，以及淘系的商品系统， UNP 的营销服务，还有支付系统，当然后面还有一些汇金平台以及各个中间件系统等从集团层面提供的业务支持。

OK，那我们接下来再往后看，这里还设置了商品营销为服务。营销其实是一个商品的子领域，它是完全围绕商品来构建的，借由营销服务来定义一些营销规则、优惠的玩法，创建优惠券、领券等等这样的不同服务。OK，再往下看商品的导购服务，导购是什么呀？导购是一个你在购物车页面、详情页面都可以给你透传一些用户推荐商品的这样一个功能，所以从这里可以看出什么呀？第一点它可以增加你的用户复购率以及运营指标，增加你的整体的收入。另外一点，它的底层的业务玩法跟其他位付大有不同，导购这里后台对接的是什么呀？机器学习平台等等。
