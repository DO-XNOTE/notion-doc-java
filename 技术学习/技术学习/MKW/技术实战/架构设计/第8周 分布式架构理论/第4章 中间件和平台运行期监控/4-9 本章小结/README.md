# 4-9 本章小结

本章小结
这一节当中的缓存三大坑是高并发场景当中需要重点关注的内容，建议同学们结合前面章节的热点 数据处理方案，加深对缓存应用的理解。平台监控是一线大厂在项目过程中作为重点环节来构建 的，线上故障的抢修和应急响应全靠这些业务埋点和预警，才能做到第一时间发现和抢修，对于小 型应用来说，如果没有晚上的监控平台也埋点框架，那么也可以从我做起，在项目关键路径上打一 些业务日志，对线上排查问题也是大有帮助。
