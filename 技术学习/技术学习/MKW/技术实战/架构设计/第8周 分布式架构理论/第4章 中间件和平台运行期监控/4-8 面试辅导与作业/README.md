那么 Kafka 是不是做不到可靠性的？其实并不是这样的， Kafka 的 delivery mode 咱前面小节当中有介绍，也是可以实现一些可靠性比较高的场景的，但是相对来说，牺牲 Kafka 的高性能，做到可靠性就有点得不偿失。那不妨在这些可靠性要求比较高的场景，下面我们转换成Rocketmq， Rabbit MQ 等等。
