# 3-10 本章小结

本章小结
这一章节主要讲了关于“数据”方面的知识点，平时大家的注意力都集中在“应用”层面，往往忽视了底 层数据层面的高可用构建。在大型互联网应用中，数据层的高可用方案是很重要的一环，而且这并 非只是“DBA”的工作，比如分库分表和数据异构方案，则是开发团队在搭建系统之初就应该去思考 的环节。
关于数据方面的其他知识点，我们将在数据库设计章节跟同学们详细介绍。
