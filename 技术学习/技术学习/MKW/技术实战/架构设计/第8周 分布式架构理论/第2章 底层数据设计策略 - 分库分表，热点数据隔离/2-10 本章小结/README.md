# 2-10 本章小结


本章小结
这一章节讲到了两个在高并发系统设计中经常运用的技术-分库分表和热点隔离，分库分表和数据异 构被广泛应用在商品、订单等领域中，用来满足这种大数据量和多维度查询的要求，热点隔离也是 大型互联网应用中必须考虑的容错方案。可能大部分同学并没有在实际项目中运用这些技术，但还 是要对这些技术方案有一定的了解。大厂面试有时也会考察候选人是否有高并发场景的经验，那这 时候这些技术方案就用得上了。
关于课程中提到的sharding-proxy和数据迁移的落地案例，我们将在后面的章节中详细讲解。
