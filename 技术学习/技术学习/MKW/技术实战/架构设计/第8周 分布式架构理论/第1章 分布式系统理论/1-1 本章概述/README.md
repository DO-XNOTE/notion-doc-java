再后面我们跟大家介绍一个 n WR 理论，以及另一个非常有趣的分布式一致性算法，叫Gosip，从这个算法的名称含义当中，同学们应该也能看出一点端倪，它是一个传播性的算法，那具体的细节我们在课程当中再跟大家详细介绍。OK，再往后我们这里要跟大家去讨论一个拜占艇将军的问题。拜占艇将军问题主要是描述了在一个分布式环境当中不可信的节点，它对集群的一致性所造成的破坏。 Raft 算法就是一个解决了半占庭将军问题的一个经典算法。 go SIP 算法就是落入了拜占庭将军问题的一个陷阱，我们在后面的小结里将会去比对这两种算法之间的不同。
