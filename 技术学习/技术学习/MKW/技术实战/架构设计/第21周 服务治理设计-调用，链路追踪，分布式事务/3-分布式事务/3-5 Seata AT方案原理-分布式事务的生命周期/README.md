那我通过分布式事务调用链去调用到的每一个微服务，它底层的这个 r m 组件，跟数据库直接打交道的 r m 组件，他们会去怎么样？你看 register branch 注册分支事务，那这一个步骤当中，它会将整个的 XID 贯穿整个微服务调用链路，也就是说我每一个分支事务实际上都挂在这同一个XID，也就是全局 ID 之下。
