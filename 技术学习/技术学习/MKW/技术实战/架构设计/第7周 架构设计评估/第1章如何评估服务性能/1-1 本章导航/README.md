那我们先来看一下本章课程的课程目录。首先我们来学习两个非常经典的软件架构评估的理论，第一个就是ATAM，第二个理论是基于 ATM 之上的一个成本效益的分析方法，那它有助于我们来去评估一个软件架构，还有一个软件解决方案的投入产出比。那国内的软件架构评估，尤其是互联网行业，它秉承的是那种乱拳打死老师傅的做派。但是在国外的公司，像湾区的一线大厂里，你做一个软件架构设计方案的评估，会经过非常复杂的非常长的一个流程来，有很多人去 review 你的方案，所以它不可能是一个拍脑袋就上的决定，它要有很多的方法论，那经过很长时间沉淀下来的这些理论基础来去支撑你的这个架构设计的方案。
