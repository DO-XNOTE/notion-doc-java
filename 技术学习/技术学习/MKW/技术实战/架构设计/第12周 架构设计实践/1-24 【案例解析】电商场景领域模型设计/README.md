那这些是什么？是通用域和支撑域，所以我们可以什么在网上买一些这种 pass 的服务，可以买些第三方的产品，也可以自研的，这些都不是我们整个企业盈利的核心。对，盈利的核心在什么？在我们的核心子域通常有买、支付、履约、卖，还有一些市场导流、用户管理、风控、客服搜索、财务相关的产品中心、定价中心、用户认证、内部的用户管理，还有推荐系统、社交系统、第三方合作、库存中心等等。这是我罗列了一些常见的我们 C2C 平台会用到的核心词语。
