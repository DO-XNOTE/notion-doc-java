我们所谓的保险原理不是说我们买的那个保险，或者用一个词就非常好去理解，它就是双重保险。所以保险它的意思就是我一个人做了一个事儿，然后我让它再做另一个事儿去，两个事儿共同起来去验证这个，它那就像密码一样，是吧？我们既要注密码又要刷脸，我们想想那些电影里面是不是都是这样子的？它通过两重密码来做到一个双保险，那自古以来的这个保险原理的案例是什么样子？我们就来看这样的一个案例，他是在明朝，马皇后生病了，然后太医就进宫为马皇后把脉，几个太医把脉之后，就会通通的关在不同的小房间里开药单，然后把他们开出来的药放到一起去对照。


如果某一种药材是几个大夫都开的，那么马皇后都吃，如果某个药材只有某个大夫开了，而别的大夫就没有，那马皇后就不吃这种药材。他的目的是什么呢？就是防止大夫被别人收买了，然后给他下毒，所以他就将几个大夫都分别开了，那针对一个每个大夫都开了的，肯定是面等于这个病去做的这样的药材。但是如果是一个单独的一个药，极有可能是毒药，所以马化后就通过这种方式来确保所有的药都是可以吃的。

其实它目的就是为了我打通零缺陷。所以在日本中，在日本的工业工程就是这种叫做日本的这样的质量管理体系里，有这样的一个词，叫做零缺陷管理。离线性管理是怎么认为的？他认为我们所有的事情都是以人为本的，人才是这个事情的最主要的一个点。那更多的在零缺陷这个点上还是以预防为主，因为你后置的去检查这种次品，它已经产生了，那已经产生损失了，因为零件也需要成本，那么通过预防来避免次品零件的产生，可以最大程度上来保证这个整个产品的一个质量，他们认为差错其实是可以避免的，并且零缺陷是可以实现的。


这个就是这个管理思路的一个出发点，然后所有的面向于这种房呆都是从这个出发点衍生出来，他们认为可以实现，所以他们就会设计各种各样的一个房贷的方法去达成这个效果。当然在工业构成传递到美国之后，美国根据自身的工业情况，制造业的情况延展出了相关的一理论。因为美国的时候是具备非常强的工业水平，以及他们的工业管理也是世界领先的，所以美国也会有自己独特的一些管理的文化，但是他们和日本的其实是存在较大差异的，所以在美式的零均线管理中，他们认为是我们可以第一次就把事情做对，因为你当你出现第一次做不对，那就意味着会有第二次，第二次也有可能做不对，所以第一次把事情做对，它就不会再出现问题。
