看一些保险的案例，比如说最常见的这个电磁轴的铜锁，它需要同时长按两个键才能打开铜锁，这样子的话，当小孩儿来瞎按的时候，他这很难能够做到。我同时按住这个两个键按很久，然后把它打开之后我再按其他的键，这种概率微乎其微，所以它就可以避免儿童误触导致烫伤。


还有一个什么呢？就是我们的汽车放手刹，汽车的是要踩着刹车你才能放电子手刹的，所以这种它去确保你一定是在刹车的状态下去设置，那它其实也是一个双重保险，就是你要踩着刹车你才能放出手刹，你光踩刹车它没有用，你直接拉手刹它也拉不下去，这样的方式就可以避免我在放手刹的时候，我的车一定是刹车的状态，它不会是在跑着的一个状态而产生手刹的一个下放，而避免车紧急的制动导致一些事故。

自动是什么呢？自动就是指这个物件它自己在达到一定条件下，或者达到一定的数值水准，然后以及一些情况的触发下，它自动的完成相关的事宜，这样子的话就不需要人去处理了。比如说像电饭锅的自动开关，电饭锅煮熟后它就会自动跳开了，而避免你持续站着导致糊，甚至导致火灾。


大家看这个电饭锅就会非常有印象，就是以前我们家里的电饭锅都是这个样子的，把它摁下去它就开始加热，它熟了之后它就自动跳起来了，就是属于保温的状态了，而不会是它摁着它还会一直加热，需要人把它调起来，那这种情况下它就不会因为过度加热而糊锅，进一步的导致火灾。


还有比如说我们的过载保护电闸，当电流过大时，开关就会自动地断开，避免电流过大失意，因为短路而导致，从而进一步的引发火灾。这样的话它就能够帮助我们当家里出现了某个短路的时候，它就能自动的去跳闸，然后保护我们家中的电器。

是指我们没有办法完全杜绝，但是我们可以通过一些方式减小其犯错误所能带来的损失，或者是通过一些方法缓和在整个执行过程中人的一些体感，而避免人犯一些错误。举一个例子，你叫高速为什么弯堵？我们会发现我们平时坐高速，它明明是说可以造成一条直线，因为两点之间直线最短，你造成直线你的用的这种建材也是最少，大家开的也会最快。但是为什么高速上还要拼命的造这么多弯呢？因为高速上它其实速度很快，那我们就很容易产生相关的疲劳，因为你周围的风景可能都是类似的，而通过建设大量的弯道，这样的话驾驶员就要一直的注意转向，大家注意向左转向右转向左转向右转，然后避免因为你失血疲劳，整个人进入一个突昏欲睡的状态，那这种一旦打了瞌睡，那这个事故就太容易发生了。


还有一个像这种压力锅的限压法，我们小时候的家里这种压力锅，它上面就会有个限压塔，它会不停地在呲呲呲呲，在放弃，它通过根据内部的压力不停地放出部分气体，保证里面的压力不会过高。而避免说你压力一直在加热，然后里面压力特别特别高，然后你一开锅就palm，你就炸掉了，通过这样的方式来缓和内部的压力，从而保障我们这个压力锅，它的爆炸的概率就大幅度下降。
