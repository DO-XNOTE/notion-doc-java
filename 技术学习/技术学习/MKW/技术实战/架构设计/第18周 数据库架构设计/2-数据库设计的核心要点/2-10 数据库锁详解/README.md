共享锁也叫 SLock, S 锁，它允许持有该锁的事物读取一行数据。而排它锁也叫 XLock， X 锁，它允许持有该锁的事物删除或者更新一行数据。这里有同学可能要说了，插入操作怎么办？我们一会儿再说。共享排他锁有这么几个特性。首先，它是一种行级锁。


第二，如果一个事物持有行 x 的共享锁，其他事物也可以获得这一行的共享锁。很好理解，因为共享锁是用来做读取的，而读取操作，它并不会改变这一行的数据。两个事物都去读取同一行数据，不会有并发问题，对吧？第三，如果一个事物持有了行 x 的排他锁，其他事物必须等待这个事物释放之后才能持有。也很好理解。从排他所它的字面意思就会看出来了，是排他的。
