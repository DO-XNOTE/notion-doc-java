## 2.1泛化

**介绍：**  
泛化(Generalization)表示类与类之间的继承关系，接口与接口之间的继承关系，或类对接口的实现关系

（1）继承**介绍：**

继承使用**空心三角形+实线**表示。**示例：**

## 2.3关联

**介绍：**   
对于两个相对独立的对象，当一个对象的实例与另一个对象的一些特定实例存在固定的对应关系时，这两个对象之间为关联关系。

**表示方法：**   
关联关系用**实线箭头**表示。
