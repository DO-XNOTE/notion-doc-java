大家好，我是大木，下面来对这一章做一个简单的总结。这一章课上介绍了各种数据库高可用方案的大致原理、适用场景、优缺点，并手把手的搭建了每一种方案。虽然内容比较多，搭建也需要一定的动手能力，但总体来说内容并不难。希望同学们至少能够掌握每一种方案的大致原理、适用场景以及优缺点，并能够根据项目的特点选择合适的高频用方案。


至于搭建的具体工作，看公司的情况。对于有 DBA 的团队，一般不会由架构师亲自去搭建，而如果数据库直接由开发团队自己维护的话，那么可能就需要亲自搭建了。好，这一章就到这里，同学们可以稍微的整理回顾一下，稍后我们将进入新章节的学习，谢谢大家。
