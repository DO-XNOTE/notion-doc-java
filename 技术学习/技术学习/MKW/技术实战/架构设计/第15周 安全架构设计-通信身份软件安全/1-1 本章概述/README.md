嗯，包括前面的渗透测试，包括这个通信安全章节的网络攻击手段，加起来基本上小伙伴就可以什么成为未来红队的成员了，一个企业里面通常是吧？在安全部分都会分为红队和蓝队，红队负责攻击，那蓝队负责什么？负责防御，所以蓝队又称为最强支盾网络防御。那蓝队有哪些传统套路呢？最经典的就是 Wharf 应用防火墙，这个防火墙跟我们平时说的防火墙可是两种不同的产品，所以通常我们在业界不称为防火墙，而直接称为瓦甫，这个防火枪是在防御应用层，是防御什么我们的 o WASP 十大攻击的，我们会重点聊一聊它如何来防，那除此以外， ITS IPS 这种入侵检测和防御也非常经典，我们会来聊一聊他如何实现的，以及他们两者之间的区别。


那另外大部分的小伙伴在家里办公的时候，是不是都使用了 VPN 接入到公司的网络啊？这个 VPN 后台也有多种实现手段，其中很经典的 IP SEC 复原，老师会从两横两纵这个矩阵式的结构来跟大家聊一聊如何实现 IP SEC。那除此以外，有些经典的加密算法会在什么在网络中使用？比如说在网络中我们怎么样用加密来实现邮件的通讯？怎么样用加密来实现 HTTPS 的隧道加密通信等等，其中尤其是 HTTPS 的隧道加密，大家如果记得没错的话，前面的服务网格是不是我们曾经实现过一个叫 mutual TLS 双向认证加密啊？这一章节我们将采用 Java 代码实战，看一看如果没有 is still 的帮助的情况下，我们应该怎么样写代码来实现隧道加密？好，有理论、有实战、有攻击、有防御，这就是本章的整体架构了。好，下一节我们先从网络的基本模型开始，跟大家娓娓道来，大家敬请期待。
